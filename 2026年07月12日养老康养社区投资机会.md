# 2026年07月12日养老康养社区房产投资分析报告

## 核心事件
**2026年7月12日，国务院印发《国民健康“十五五”规划》，明确提出支持社会力量兴办医养结合机构，并建设一批具备“健康驿站”功能的康养社区。**

## 未来变化预测
未来 5–10 年，传统的刚需住房将进一步退潮，而具备“深度适老化改造”与“专业医疗配套”的康养社区将成为房地产市场的结构性亮点。随着第一代中产阶级步入深度老龄化，高品质的养老房产将从“消费品”转变为“稀缺资源”。

## 影响分析
| 维度 | 影响描述 |
| :--- | :--- |
| **经济** | 养老地产将成为房地产行业转型的核心方向，拉动万亿级适老化装修与服务消费。 |
| **产业** | 整合物业管理、医疗护理、康复器材与营养餐饮，形成闭环的康养产业链。 |
| **社会** | 缓解家庭养老压力，提升老年群体的尊严感与生活质量。 |
| **国际** | 借鉴并超越日本、德国的康养模式，形成具有中国特色的“医养结合”标准。 |
| **政策** | 预计将出台康养房产契税减免、土地供应倾斜及医保进社区等支持政策。 |
| **资产价格** | 核心城市（如上海、深圳、杭州）的品牌康养公寓租金回报率将显著高于普通住宅。 |

## 投资机会判断
**是否存在可落地的投资机会：是。** 核心城市高品质康养社区的置换与持有具备长期价值。

## 投资对象描述
*   **核心城市（上海/杭州）康养公寓**：主要产品为位于成熟城区、周边1公里内有三甲医院、内部配备紧急呼叫系统及专业护理团队的高端适老化住宅。

## 投资路径及决策结论
### 个人房产买卖投资
*   **入场逻辑**：在老龄化程度高、医疗资源集中的一二线城市，置换或购入具备专业康养配套的房产。这类资产具备极强的抗跌性与租金溢价能力。
*   **风险点**：政策对康养地产定义的变动、物业管理水平下滑、流动性相对较低。
*   **交易决策结论**：
    *   **入场价格：以该区域普通住宅均价的 1.1 - 1.2 倍作为入场基准；**
    *   **退出价格：当资产租金收益率低于 3% 或 区域同类竞品饱和时退出；**
    *   **核心操作标准：必须考察物业是否具备“医疗执业许可证”，且社区内适老化设施（如无障碍通道、紧急拉绳）必须通过三方机构认证。**
