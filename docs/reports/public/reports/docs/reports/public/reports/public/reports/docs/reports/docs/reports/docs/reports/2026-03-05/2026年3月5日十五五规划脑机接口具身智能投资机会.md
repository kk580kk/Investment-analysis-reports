# 2026年3月5日 十五五规划·脑机接口与具身智能投资机会

> **报告日期**：2026年3月6日  
> **信息来源**：仅检索2026年3月5日当日发布新闻

## 一、核心事件

2026年3月5日，十四届全国人大四次会议开幕，国务院总理李强发表政府工作报告，正式发布《中华人民共和国国民经济和社会发展第十五个五年规划纲要草案（2026—2030年）》。报告首次将**脑机接口**写入政府工作报告，与量子科技、具身智能、未来能源、6G并列为国家级未来产业，标志着脑机接口从实验室研究正式上升至国家战略高度。与此同时，"人工智能+"连续第三年写入报告，并新增"智能体"、"智能经济"等全新表述，政府工作报告明确提出"深化拓展'人工智能+'，促进新一代智能终端和智能体加快推广，推动重点行业领域人工智能商业化规模化应用"。

---

## 二、未来变化预测（5—10年结构性改变）

**技术商业化加速**：脑机接口进入政府工作报告，意味着国家将在临床准入、监管标准、资金投入三个维度全面加速。预计2026—2030年间，国内脑机接口市场规模将从当前不足10亿元扩张至百亿元量级，全球市场到2034年有望增长至124亿美元。

**具身智能进入量产元年**：2026年春晚已成为国产人形机器人系统级验证舞台，多家整机厂商加大资本开支推进量产节奏。中国人形机器人市场预计从2025年的167亿元增长至2030年的861亿元，年复合增速超过38%。

**产业链国产替代深化**：在美国持续技术封锁背景下，集成电路、谐波减速器、伺服系统等核心部件的国产化率将从当前30%—50%提升至70%以上，核心供应商将迎来业绩拐点。

## 四、投资机会判断

**存在明确的可落地投资机会**。脑机接口首次写入政府工作报告属于政策催化的一次性重大事件，历史上类似事件（如2024年"人工智能+"首次写入报告）均带来相关板块持续6—18个月的超额收益。具身智能则处于从技术验证向量产落地的关键拐点，核心零部件供应商的业绩将在2026—2027年出现实质性兑现。

---

## 五、投资对象描述

### 标的一：绿的谐波（688017）
**主要产品/业务**：谐波减速器（机器人关节核心部件），占国内市场份额约35%，是人形机器人、工业机器人关节传动的"卡脖子"零部件。2025年业绩预增104%—131%，进入业绩兑现期。

### 标的二：汇川技术（300124）
**主要产品/业务**：工业自动化核心部件（伺服系统、变频器、PLC），同时布局机器人控制系统，是机器人ETF第一大重仓股（占比9.71%）。公司是国内工业自动化领域的绝对龙头，受益于具身智能和制造业升级双重驱动。

### 标的三：拓普集团（601689）
**主要产品/业务**：机器人核心部件（一体化压铸、底盘系统），同时是特斯拉人形机器人核心供应商，具备全球化供应链优势。

### 标的四：三博脑科（301293）
**主要产品/业务**：神经外科专科医院连锁运营（北京、深圳等地），是国内脑机接口临床应用的领先机构，具备脑机接口临床试验的天然场景优势。

## 六、长期趋势可行性投资路径

### 路径一：具身智能核心零部件（优先级：★★★★★）

**方向**：布局人形机器人核心零部件供应商，把握量产元年的业绩兑现行情。

**标的类型**：A股上市公司（深证/上证）

**产品/业务描述**：谐波减速器（绿的谐波）、伺服系统（汇川技术）、机器人核心部件（拓普集团）——三者构成人形机器人"关节+驱动+结构"的完整零部件体系。

**入场逻辑**：十五五规划将具身智能列为六大未来产业之一，国家大基金首次出手具身智能（银河通用获25亿融资），2026年春晚验证了国产人形机器人的系统级能力。绿的谐波2025年业绩预增超100%，进入业绩兑现期，估值有支撑。

**风险点**：人形机器人量产进度不及预期；核心客户（特斯拉）订单波动；美国对华技术出口管制升级可能影响部分精密制造设备进口。
