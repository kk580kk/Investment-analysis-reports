### 2. 分众传媒 (002027) - 详细分析

#### 2024年财务表现
- 营业收入：122.62亿元
- 归属于上市公司股东净利润：51.55亿元
- ROE：长期保持在40%以上的高水平
- 分红政策：每10股派发现金红利2.30元

#### 商业模式深度解析
**"二房东"模式的独特优势**
1. **上游租赁**：向物业公司租赁电梯媒体点位
2. **下游出租**：向广告主出租媒体使用权
3. **规模效应**：点位数量越多，单位成本越低，议价能力越强

#### 三重护城河分析
**1. 基于先占优势的低成本点位（供给侧护城河）**
- 早期进入市场，获得优质点位的长期租赁合同
- 后来者面临更高的点位获取成本
- 物业方更愿意与成熟运营商合作

**2. 基于规模效应的成本优势（运营侧护城河）**
- 全国性网络布局带来规模经济
- 统一采购、运营、维护降低边际成本
- 技术平台投入可在更大规模上摊销

**3. 基于网络效应的客户粘性（需求侧护城河）**
- 覆盖面广的媒体网络对广告主更有价值
- 客户投放习惯形成后转换成本高
- 品牌广告主重视媒体的稳定性和可靠性

#### 竞争格局分析
- **市场地位**：电梯媒体领域绝对龙头，市占率超过70%
- **竞争对手**：新潮传媒等，但规模差距明显
- **行业壁垒**：高资本投入、点位资源稀缺、运营经验要求高

#### 投资价值评估
**优势**
1. 独特的媒体场景，无法被线上媒体完全替代
2. 覆盖中高收入人群，广告价值高
3. 现金流稳定，分红能力强
4. 护城河深厚，竞争优势明显

**风险**
1. 宏观经济波动影响广告投放
2. 新媒体形式的冲击
3. 点位租金上涨压力

## 估值计算与投资建议

### 估值方法论
基于投资大师的估值理念，我们采用多种估值方法进行交叉验证：

#### 1. 格雷厄姆估值法
**公式**：内在价值 = EPS × (8.5 + 2G)
其中G为预期增长率

#### 2. 巴菲特DCF估值法
**核心要素**：
- 自由现金流预测
- 永续增长率假设
- 折现率选择

#### 3. 彼得·林奇PEG估值法
**公式**：PEG = PE / 增长率
- PEG < 0.5：被严重低估
- PEG < 1.0：合理或低估
- PEG > 1.0：可能高估
