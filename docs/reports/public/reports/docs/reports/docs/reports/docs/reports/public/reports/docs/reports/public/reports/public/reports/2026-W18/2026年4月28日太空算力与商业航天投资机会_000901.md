# 2026年4月28日太空算力与商业航天投资机会 000901.md

**核心事件**：2026年4月27日，国家航天局召开商业航天高质量发展圆桌会议，明确要求前瞻布局太空算力、太空制造等新业态。同日，成都ADAspace公司宣布其“星算”计划（2800颗算力卫星）已完成全球首次在轨部署通用大模型，并推出全球首个太空算力云服务平台 [1] [2]。

**未来变化预测**：未来5–10年，人类算力基础设施将从“陆海”向“空天”实现维度跃迁。太空算力将成为继地面数据中心之后的全新战略高地，彻底解决地面算力面临的能耗、散热及地理覆盖盲区问题。到2035年，全球太空经济规模预计达1.8万亿美元，中国将建成千星规模的太空算力网络，实现全球无死角的智能服务覆盖，重塑自动驾驶、远洋探测、应急救援等领域的底层逻辑 [3]。

**影响分析**：
*   **经济维度**：太空算力产业将催生万亿级的新兴市场，带动火箭发射、卫星制造、星载芯片、激光通信等上下游产业链的爆发式增长，成为拉动经济高质量发展的新引擎。
*   **产业维度**：商业航天将从“拼发射”进入“拼应用”的新阶段。传统的卫星通信将向“通算一体化”演进，星载AI芯片、抗辐照材料、高效热控系统等核心技术将迎来密集突破期。
*   **社会维度**：太空算力的普及将极大提升偏远地区、海洋及灾害现场的应急响应能力，缩小全球数字鸿沟，实现真正的“算力普惠”。
*   **国际维度**：太空算力将成为大国科技博弈的新焦点。中国通过前瞻布局，有望在太空算力标准的制定、轨道资源的抢占以及全球服务的提供上占据主导地位。
*   **政策维度**：国家层面将持续出台支持商业航天发展的政策，如开放更多发射工位、提供专项资金支持、完善商业航天法律法规等，为产业发展保驾护航。
*   **资产价格维度**：具备核心技术壁垒的商业航天企业、星载芯片设计公司及卫星制造企业将迎来长期的估值提升，成为资本市场追逐的“硬科技”核心资产。

**投资机会判断**：存在高度确定且可落地的长期投资机会。太空算力已从概念验证走向商业化落地，国家航天局的定调标志着产业发展进入快车道。随着火箭发射成本的持续下降和卫星量产能力的提升，相关产业链的业绩爆发期即将到来。

**投资对象描述**：
*   **标的类型**：上证/深证/港股股票投资
*   **主要产品/业务**：航天科技（000901.SZ）是中国航天科工集团旗下的核心企业，主要从事航天应用产品、汽车电子产品、车联网及工业物联网等业务。公司在航天应用领域拥有深厚的技术积累，其产品广泛应用于各类航天器及卫星系统，是商业航天产业链的重要参与者。

**交易决策结论**：
买入：航天科技（股票代码000901），买入价格21.11元。
*   **入场逻辑**：航天科技作为航天科工集团的重要上市平台，深度受益于国家商业航天战略的推进。当前股价为21.11元，处于历史相对低位（52周最高35.40元）。随着太空算力、卫星互联网等新业态的爆发，公司航天应用产品的订单有望迎来大幅增长。同时，公司在车联网等领域的布局也为其提供了多元化的增长动力。
*   **风险点**：商业航天项目投资大、周期长，存在技术研发失败或发射失利的风险；市场竞争加剧导致产品毛利率下降；宏观经济波动影响公司汽车电子等民品业务的需求。
