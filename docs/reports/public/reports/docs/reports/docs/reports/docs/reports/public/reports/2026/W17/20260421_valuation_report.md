### 格雷厄姆内在价值法
- 基于格雷厄姆经典公式: 内在价值 = EPS × (8.5 + 2G) × 4.4 / 债券收益率

- 安全边际 = (内在价值 - 当前价格) / 内在价值
- 建议安全边际 ≥ 33%
