## 风险管理策略

### 仓位控制
- **总仓位**: 根据市场环境动态调整
- **单只股票**: 不超过总资产的15%
- **单个行业**: 不超过总资产的30%

### 止损策略
- **技术止损**: 跌破重要技术支撑位
- **基本面止损**: 基本面出现重大恶化
- **时间止损**: 持有期超过预期但未达到目标

### 分散投资
- **股票数量**: 投资组合包含10-20只股票
- **行业分散**: 投资至少3-5个不同行业
- **市值分散**: 大中小盘股合理配置

## 择时策略

### 买入时机
- 市场PE低于历史20分位数
- 政策底部确认
- 技术面出现明确反转信号

### 卖出时机
- 市场PE超过历史80分位数
- 货币政策明显收紧
- 技术面出现明确顶部信号

## 重要风险提示

1. **市场风险**: 股市有风险，投资需谨慎
2. **政策风险**: 关注监管政策变化
3. **流动性风险**: 注意市场流动性变化
4. **个股风险**: 分散投资降低个股风险
