# 2026年6月8日全球AI资本开支潮及中际旭创投资机会 300308

## 1. 核心事件
2026年6月7日消息，全球四大云巨头（微软、谷歌、Meta、AWS）2026年合计AI资本开支规划超7000亿美元，光模块产业链处于1.6T/3.2T技术迭代拐点，中际旭创单日成交额创历史纪录。

## 2. 未来变化预测
未来5-10年，算力将成为全球经济的“新电力”。AI从“模型竞赛”转向“基础设施大基建”，1.6T及更高速率的光模块将成为全球数据中心的标准配置，中国光模块厂商凭借成本与技术迭代优势将占据全球80%以上的市场份额。

## 3. 影响分析 (MECE规则)

| 维度 | 影响分析 |
| :--- | :--- |
| **经济** | AI算力投入成为拉动GDP的新引擎，全社会全要素生产率因AI普及而提升。 |
| **产业** | 光通信行业进入高频迭代期，从可插拔向CPO（共封装光学）演进。 |
| **社会** | 算力普惠化推动智能服务渗透至生活每个角落。 |
| **国际** | 算力霸权成为国家竞争的核心，算力中心布局成为地缘政治博弈点。 |
| **政策** | 各国加大对AI基础设施的补贴与国家级算力网建设支持。 |
| **资产价格** | 算力龙头估值中枢由“成长股”转向“价值蓝筹”，现金流大幅改善。 |

## 4. 投资机会判断
**存在明确可执行投资机会。** 尽管短期成交额巨大导致股价波动，但长期逻辑（7000亿美元资本开支）极其坚固。

## 5. 投资对象描述
**中际旭创 (300308.SZ)**：全球光模块领军企业，在800G和1.6T领域拥有绝对的技术领先地位和全球份额，是英伟达、谷歌等巨头的核心供应商。
**天孚通信 (300394.SZ)**：全球光器件领先厂商，受益于光模块产能扩张带来的上游器件需求爆发。

## 6. 投资决策结论
*   **买入：中际旭创 (300308)**
    *   **买入价格**：1150.00元人民币以内（近期大跌提供入场机会）。
    *   **入场逻辑**：AI资本开支无缩减迹象，1.6T放量在即。
    *   **风险点**：技术路线切换超预期（如硅光替代）；贸易制裁升级。

*   **买入：天孚通信 (300394)**
    *   **买入价格**：380.00元人民币以内。
    *   **入场逻辑**：作为光模块的上游，其竞争格局更优，毛利率稳定。
    *   **风险点**：下游客户订单波动风险。
