# 2026年5月3日 中国对非洲全面零关税投资机会

## 一、核心事件

2026年5月1日起（新闻于5月3日广泛发酵），中国正式对53个非洲建交国全面实施100%税目产品零关税，成为全球首个对所有非洲建交国和建交最不发达国家实现单方面、全覆盖零关税待遇的主要经济体。此前，中国已于2024年12月1日对33个非洲最不发达国家实施100%税目零关税，此次是将范围扩大至全部53个建交国。

## 二、未来变化预测（5-10年）

未来5至10年，中非贸易结构将发生根本性改变。非洲将成为中国重要的资源供应地与初级产品加工基地，中国将深度整合非洲产业链，形成"中国资本+非洲资源/劳动力"的新型全球化分工模式。中非双边贸易规模有望从当前水平翻倍，非洲矿产、农产品、能源对华出口将大幅增长。

## 三、影响分析（MECE框架）

**经济维度：** 此举将大幅扩大中非双边贸易规模，有效缓解中国部分资源与农产品的进口压力，并为中国企业在非洲的投资布局提供更有利的贸易环境。

**产业维度：** 国际贸易、港口物流、跨境电商以及在非洲布局矿产/农业的企业将获得巨大增量空间。大宗商品供应链管理、贸易融资等细分领域将迎来业务爆发。

**社会维度：** 经贸往来将促进中非民间交流，提升非洲当地就业与生活水平，进一步深化中非命运共同体建设。

**国际维度：** 这将进一步巩固中国在"全球南方"的领导地位，对冲欧美贸易壁垒带来的出口压力，在中美博弈背景下为中国开辟更广阔的战略纵深。

**政策维度：** 标志着中国高水平对外开放向发展中国家倾斜，致力于构建更加包容的国际贸易新秩序，为中国的多边主义外交提供有力支撑。

**资产价格维度：** 深度参与中非贸易的供应链综合服务商及港口物流企业资产价值将面临重估，相关公司的业绩增速有望超越市场预期。

## 四、投资机会判断

**存在可落地、可执行的投资机会。** 零关税政策将直接刺激中非进出口贸易量激增，具备成熟非洲贸易渠道和供应链管理能力的企业将迎来业绩爆发。

## 五、投资对象描述

厦门国贸（600755.SH）是中国领先的综合性供应链服务商，主营业务为供应链管理（大宗商品贸易，包括矿产、农产品、化工品等）及金融服务。公司在矿产、农产品等大宗商品进出口领域拥有强大的全球网络和深厚的运营经验，是中非大宗商品贸易的重要参与者。2026年一季度营收869.01亿元，同比增长23.20%，出海提速趋势明显。
