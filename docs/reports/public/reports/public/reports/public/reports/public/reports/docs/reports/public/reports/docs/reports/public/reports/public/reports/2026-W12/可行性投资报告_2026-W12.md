## 一、投资理论基础

### 1.1 格雷厄姆价值投资理论

本杰明·格雷厄姆被誉为"价值投资之父"，其核心理念是寻找市场价格显著低于内在价值的股票，通过"安全边际"降低投资风险。

> "投资操作是以深入分析为基础，确保本金安全，并获得适当的回报。不满足这些条件的操作就是投机。"  
> ——本杰明·格雷厄姆，《证券分析》

**核心选股标准（7项）：**

| 指标 | 标准 | 说明 |
|------|------|------|
| 市盈率（PE） | ≤ 15 | 估值合理，避免高估值风险 |
| 市净率（PB） | ≤ 1.5 | 资产价值被低估 |
| PE × PB | ≤ 22.5 | 综合估值指标 |
| 流动比率 | ≥ 2.0 | 短期偿债能力充足 |
| 资产负债率 | ≤ 50% | 债务风险可控 |
| 净资产收益率（ROE） | ≥ 15% | 盈利能力达标 |
| 利润增长率 | ≥ 3% | 基本成长性要求 |

**内在价值公式**：内在价值 = EPS × (8.5 + 2G) × 4.4 / AAA债券收益率，其中 G 为预期年增长率。

### 1.2 巴菲特质量成长理论

沃伦·巴菲特在继承格雷厄姆价值投资理念的基础上，发展出了独特的质量成长投资理论，更注重企业的"护城河"——即持续竞争优势。

> "以合理的价格买入优秀的企业，远胜于以便宜的价格买入普通的企业。"  
> ——沃伦·巴菲特

**护城河四大类型**：无形资产（品牌、专利）、转换成本、成本优势（规模效应）、网络效应。

**核心选股标准（7项）：**

| 指标 | 标准 | 说明 |
|------|------|------|
| 净资产收益率（ROE） | ≥ 20% | 超强盈利能力 |
| 毛利率 | ≥ 40% | 定价权与竞争优势 |
| 净利率 | ≥ 10% | 盈利效率 |
| 营收增长率 | ≥ 7% | 稳健增长 |
| 利润增长率 | ≥ 10% | 利润增长 |
| 资产负债率 | ≤ 50% | 财务稳健 |
| 投入资本回报率（ROIC） | ≥ 15% | 资本效率 |

### 1.3 彼得·林奇成长股理论

彼得·林奇在管理富达麦哲伦基金期间创造了年均 29.2% 的惊人回报率，其理论强调从生活中发现投资机会，寻找"十倍股"。

> "投资于你所了解的行业，并且只在你真正理解的时候才买入。"  
> ——彼得·林奇，《彼得·林奇的成功投资》

**PEG 估值方法**：PEG = 市盈率 / 预期增长率

| PEG 区间 | 估值判断 |
|----------|----------|
| < 0.5 | 严重低估，最佳买入 |
| 0.5 – 1.0 | 合理估值区间 |
| 1.0 | 公允价值 |
| > 1.0 | 高估，需谨慎 |
| > 2.0 | 严重高估，应避免 |

**核心选股标准（6项）：**

| 指标 | 标准 | 说明 |
|------|------|------|
| PEG | ≤ 1.0 | 估值与成长匹配 |
| 利润增长率 | 10%–50% | 合理成长区间 |
| 资产负债率 | ≤ 30% | 财务保守 |
| 市值 | ≤ 5,000亿元 | 中等市值，成长空间大 |
| ROE | ≥ 15% | 盈利能力 |
| 行业地位 | 细分市场领先 | 定性评估 |

## 二、市场环境分析

### 2.1 中国A股市场概况

中国A股市场作为全球第二大股票市场，截至2026年3月，共有上市公司超过5,400家，总市值超过80万亿元人民币。市场包括上海证券交易所和深圳证券交易所两个主要交易所，覆盖主板、创业板、科创板等多个板块。

注册制改革全面推进提高了市场化程度，外资持续流入，A股纳入 MSCI、富时罗素等国际指数，国际化程度不断提升。

### 2.2 当前市场特征

当前A股市场呈现出以下显著特征：

**结构性分化明显**：优质龙头企业与普通企业的估值差距不断扩大，市场资金向头部企业集中，"强者恒强"趋势日益明显。

**机构化程度提升**：公募基金、私募基金、保险资金、外资等机构投资者占比不断上升，价值投资理念得到更多认同。

**行业轮动加快**：新兴产业如新能源、人工智能、生物医药等受到市场追捧，传统行业中的优质企业也获得重新估值。

### 2.3 宏观经济与政策环境

中国经济正处于高质量发展阶段，消费升级趋势明显，数字化转型加速。货币政策保持稳健，财政政策积极有为，减税降费等措施有利于企业降低成本、提升盈利能力。

国家大力支持科技创新、绿色发展、数字经济等新兴领域，相关企业将受益于政策红利。

## 三、选股方法论

### 3.1 多策略评估体系

本报告构建了基于三大投资理论的多策略评估体系，通过量化指标和定性分析相结合的方式对股票进行全面评估。

**评分机制**：每个策略的满分为 100%，对应该策略的通过率。单项指标采用二元评分（符合/不符合），最终综合评分为三个策略得分的加权平均。

**综合评分公式**：综合评分 = (格雷厄姆得分 × 35% + 巴菲特得分 × 40% + 林奇得分 × 25%)

### 3.2 筛选流程

筛选流程分为三个阶段：

1. **初步筛选**：剔除 ST 股票、退市风险股票、数据缺失严重的股票，形成基础股票池
2. **详细分析**：对基础股票池逐一进行三大策略评估，计算综合得分
3. **最终确认**：对候选股票进行深度研究，结合定性分析确定推荐名单

### 附录B：主要财务指标计算方法

- **PE** = 股价 / 每股收益
- **PB** = 股价 / 每股净资产
- **ROE** = 净利润 / 平均股东权益
- **毛利率** = (营业收入 − 营业成本) / 营业收入
- **净利率** = 净利润 / 营业收入
- **资产负债率** = 总负债 / 总资产
- **流动比率** = 流动资产 / 流动负债
- **ROIC** = 税后经营利润 / 投入资本
- **PEG** = 市盈率 / 利润增长率
