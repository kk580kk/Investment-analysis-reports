# 2026年04月30日全球AI资本开支爆发与光模块投资机会 300308

## 一、 核心事件

2026年4月30日，谷歌、微软、Meta、亚马逊四大科技巨头相继发布财报，其AI资本开支合计接近7000亿美元，远超市场预期，显示出科技巨头在AI基础设施建设上的空前投入。其中，谷歌单季AI资本开支达到170亿美元，微软达到160亿美元，Meta和亚马逊亦大幅提升AI基础设施预算 [1]。这一数据意味着全球AI算力基础设施建设进入加速阶段，对高速光模块的需求将呈现爆发式增长。

## 二、 未来变化预测

未来5–10年，AI大模型的演进对算力的需求将呈指数级增长，数据中心内部及数据中心之间的互联带宽需求将持续提升，推动光模块从400G向800G乃至1.6T迭代升级。全球光模块市场规模预计将从2025年的约200亿美元增长至2030年的超过500亿美元，复合年增长率超过20%。中国光模块企业凭借成本优势和技术积累，将在全球市场中占据更重要的地位。

## 三、 影响分析

| 维度 | 影响分析 |
| :--- | :--- |
| **经济** | AI基础设施投资将带动半导体、光器件、服务器等上下游产业链的全面繁荣，成为全球经济增长的新引擎。 |
| **产业** | 光模块行业将迎来技术迭代与市场扩容的双重驱动，具备800G/1.6T产品量产能力的企业将获得超额市场份额。 |
| **社会** | AI基础设施的完善将加速AI在各行业的渗透应用，提升社会生产效率，同时对劳动力结构产生深远影响。 |
| **国际** | 中国光模块企业在全球市场的份额持续提升，将成为中国科技产业参与全球竞争的重要名片。 |
| **政策** | 国内"算力网"建设政策将为国内光模块企业提供额外的本土市场需求支撑。 |
| **资产价格** | 光模块龙头企业的业绩将持续高增长，估值有望随业绩兑现而维持高位，甚至进一步扩张。 |

## 四、 投资机会判断

存在高度可落地的投资机会。全球AI资本开支的爆发是光模块需求增长的直接驱动力，且这一趋势具有高度确定性。中际旭创作为全球光模块龙头，深度绑定海外科技巨头，其业绩增长与全球AI资本开支高度正相关。公司一季度业绩大幅超预期（实际EPS 5.15元 vs 预期3.11元），验证了其强大的执行力和市场地位。

## 五、 投资对象描述

**中际旭创（300308.SZ）**：全球领先的光模块制造商，产品覆盖100G至1.6T全系列，深度绑定谷歌、微软、亚马逊等全球科技巨头。公司在高速光模块领域具有技术领先优势，是全球AI数据中心建设的核心供应商。截至4月30日，公司股价报857.50元，市值约9549亿元，市盈率（TTM）63.71倍，预测市盈率（FY26）约34倍，分析师平均目标价873.33元，最高目标价1187元 [2]。2026年一季度营收195亿元，净利润57.2亿元，业绩大幅超预期。

## 六、 投资路径与交易决策

*   **方向**：全球AI基础设施建设中的光互联核心器件
*   **标的类型**：深圳创业板股票
*   **入场逻辑**：全球科技巨头AI资本开支持续超预期，中际旭创作为全球光模块龙头，是最直接的受益标的。公司一季度业绩大幅超预期，验证了需求的强劲。当前预测市盈率约34倍，对于一家年利润增速超过100%的企业，估值具有合理性。公司正积极推进港股上市，有望进一步打开估值空间。
*   **风险点**：美国出口管制政策收紧影响海外订单；800G/1.6T产品良率爬坡不及预期；竞争对手产能快速扩张导致价格战。
*   **交易决策结论**：
    **买入：中际旭创（股票代码300308），买入价格800-860元区间**（当前股价857.50元，建议在800-860元区间分批建仓，中长线目标价1187元，对应约12个月持有期）。
