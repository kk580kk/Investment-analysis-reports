# 2026年4月25日国产算力替代投资机会

## 一、 核心事件

未来5–10年，中国AI算力产业链将发生结构性改变。国产大模型的大规模采用将为国产算力芯片提供真实的压力测试场景和持续的需求拉动，推动中国本土AI产业链从“可用”向“好用”跨越。预计到2029年，中国AI芯片市场规模将从2024年的1425亿元激增至1.34万亿元，年均复合增长率达53.7%。中国将逐步建立起更具韧性、自给自足的AI生态系统，打破对英伟达等海外高端算力的依赖。
