# 2026年5月21日中油资本投资机会 000617

## 一、公司概况

中油资本（000617.SZ）是中石油集团旗下的全方位金融业务平台，旗下涵盖昆仑银行、昆仑信托、昆仑金融租赁、中意人寿、中意财险、昆仑保险经纪、中油财务等金融机构。核心产品：跨境贸易结算支付、供应链金融、产业投融资。2026年一季度营收73.81亿元，同比下降17.24%（因利息收入确认节奏波动）；归母净利润15.73亿元，同比上升22.89%，盈利能力持续改善。

## 二、投资逻辑

### 核心催化剂：中俄本币结算扩大的直接受益标的

普京访华确认《中俄睦邻友好合作条约》延期，核心成果之一为推进本币结算。中油资本旗下的昆仑银行是中国与俄罗斯及伊朗间资金结算的核心通道，承担着中俄贸易（2025年2279亿美元、2026年前4月852亿美元）的大量美元替代结算业务。随着本币结算比例从当前约50%向80%+提升，昆仑银行跨境结算规模有望翻倍。

### 基本面支撑

1. **金融牌照齐全**：集齐银行、信托、租赁、保险、证券等全金融牌照，具备一体化金融服务能力
2. **利润增速强劲**：一季度归母净利润15.73亿元，同比增22.89%，扣非净利润同比增20.05%
3. **机构目标价溢价**：90天内机构目标均价10.34元，当前8.62元仍有约20%上行空间
4. **政策支撑**：CIPS（人民币跨境支付系统）交易量持续爆发，中油资本作为CIPS参与方受益

## 三、交易决策结论

**买入：中油资本（000617），买入价格8.20-8.80元/股**

- **买入依据**：中俄本币结算扩大已从概念进入实质推进阶段。昆仑银行2025年跨境结算量同比增长约35%，2026年增速有望进一步提升。当前PB约0.8倍，低于银行板块平均PB的1.0倍，具备估值修复空间。
- **目标价格**：10.30-11.00元/股（对应机构目标均价及PB修复至1.0倍以上）
- **持有周期**：中长期（12个月）
- **止损价格**：7.50元/股
