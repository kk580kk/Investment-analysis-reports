# 2026年04月29日内需扩大投资机会 688041

## 一、 核心事件
2026年4月28日，中共中央政治局召开会议，分析研究当前经济形势和经济工作。会议强调“以更大力度和更实举措抓好经济工作”，并提出“加强水网、新型电网、算力网、新一代通信网、城市地下管网、物流网等规划建设”，全面实施“人工智能+”行动，深入挖掘内需潜力 [1] [2]。

## 二、 未来变化预测
未来5–10年，中国经济增长的驱动力将加速从传统的房地产和基建投资转向以“六网”建设（特别是算力网和新型电网）为代表的新型基础设施投资，以及由“人工智能+”带动的服务业扩能提质和消费升级。内需的结构性优化将成为实现“十五五”良好开局的关键支撑 [3]。

## 三、 影响分析
| 维度 | 影响分析 |
| :--- | :--- |
| **经济** | 通过“六网”建设等重大工程项目牵引，带动有效投资，同时通过新型基础设施建设带动产业升级，发挥对消费的促进作用，实现稳投资和促消费的良性循环。 |
| **产业** | 算力网、新型电网等新基建产业链将迎来政策红利期，相关设备制造、工程建设、运营服务企业订单有望大幅增长。 |
| **社会** | “人工智能+”行动将催生智能原生新业态新模式，提升公共服务效率，改善居民生活体验，同时对劳动者技能提出更高要求。 |
| **国际** | 强大的国内市场和完善的新型基础设施将增强中国经济的韧性，提升在全球产业链供应链中的核心地位，吸引更多高质量外资。 |
| **政策** | 预计将出台更多配套的财政和货币政策，如专项债发行加速、结构性货币政策工具发力，以保障重大工程项目的资金需求。 |
| **资产价格** | 算力基础设施、智能电网、人工智能应用等相关板块的上市公司将迎来业绩与估值的双重提升。 |

## 四、 投资机会判断
存在高度可落地的投资机会。政治局会议明确了“算力网”和“人工智能+”的战略地位，这为国产算力基础设施建设提供了强有力的政策背书。具备核心技术、能够深度参与国家级算力网络建设的企业将获得确定性的成长空间。

## 五、 投资对象描述
**海光信息（688041）**：国内领先的高端处理器提供商，主营业务涵盖CPU和DCU（深度计算处理器）。其产品在电信、金融、互联网等关键行业实现了广泛应用。公司已打造出全栈软硬件协同体系，并与主流大模型完成全面适配。2025年营收143.76亿元，同比增长56.91%，业绩持续高增长 [4] [5]。

## 六、 投资路径与交易决策
*   **方向**：算力网建设与国产CPU/DCU
*   **标的类型**：上证科创板股票
*   **入场逻辑**：政治局会议定调“算力网”建设，海光信息作为国产CPU和DCU的双龙头，是国家算力基础设施建设的核心供应商。其产品性能优异，生态兼容性好，业绩兑现度高。近期股价稳步上行，主力资金持续净流入，处于上升通道。
*   **风险点**：上游晶圆代工产能受限；市场竞争加剧导致产品毛利率下降；研发进度不及预期。
*   **交易决策结论**：
    **买入：海光信息（股票代码688041），买入价格285-295元区间**（结合近期股价在296元附近，逢低买入，中长线目标看至350元以上）。
