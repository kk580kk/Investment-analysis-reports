# 2026年06月11日智能经济新形态投资机会 9888.HK

## 核心事件
2026年6月10日，政府工作报告首提“打造智能经济新形态”，将其列为培育壮大新动能的重点举措。国家发改委随后表示将研拟储备一批针对性、操作性强的政策工具，标志着人工智能从“技术爆发期”正式进入“产业落地红利期”。

## 未来变化预测
未来5-10年，智能经济将重构生产、分配、交换与消费全流程。人工智能核心产业规模预计将从2025年的1.2万亿元增长至2030年的10万亿元以上。AI将从独立的通信设备演变为连接并协调消费者生活全场景的智能平台。

## 影响分析（MECE规则）
| 维度 | 影响描述 |
| :--- | :--- |
| **经济** | 生产效率发生结构性跃迁，AI对GDP的贡献率将显著提升。 |
| **产业** | 传统产业全面“AI+”化，催生跨界业态重组（如AI驱动的自动驾驶、智慧能源）。 |
| **社会** | 劳动力结构转型，对具备AI协作能力的复合型人才需求激增。 |
| **国际** | 全球AI治理规则博弈加剧，中国在智能终端和智能体推广方面力争领先。 |
| **政策** | 预计将出台大规模的算力基础设施补贴及AI应用场景开放政策。 |
| **资产价格** | 具备核心大模型能力及算力储备的头部企业将迎来估值重塑。 |

## 投资机会判断
**存在可落地的投资机会。** 百度作为国内AI大模型的领军企业，其智能云业务已进入爆发期，2026年Q1增长达79%，且AI业务营收占比向50%迈进。

## 投资对象描述
*   **投资对象：** 百度集团-SW（9888.HK）
*   **主要产品/业务：** 文心一言大模型、百度智能云（AI Cloud）、Apollo自动驾驶平台、AI驱动的搜索及移动生态。

## 交易决策结论
*   **买入：** 百度集团-SW（9888.HK）
*   **买入价格：** 150.0 - 155.0 港元（当前价155.0港元，建议在150港元附近分批建仓，结合技术面支撑位及AI业务高增速预期）。
*   **卖出：** 百度集团-SW（9888.HK）
*   **卖出价格：** 190.0 港元（对应2026年目标价，预期收益率约25%）。

## 风险点
1.  算力受限风险：高端芯片进口限制可能影响大模型迭代。
2.  市场竞争：互联网巨头及AI新势力在应用层的激烈竞争。
3.  监管政策：AI安全及隐私保护政策可能带来的合规成本增加。
