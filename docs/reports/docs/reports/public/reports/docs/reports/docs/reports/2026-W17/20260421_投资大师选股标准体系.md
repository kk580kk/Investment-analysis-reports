# 投资大师选股标准体系

## 概述

本文档基于格雷厄姆价值投资、巴菲特质量成长、彼得·林奇成长股等投资大师理论，制定了一套完整的A股选股标准体系。每套策略都有明确的量化指标和评分标准。

## 一、格雷厄姆价值投资策略标准

### 1.1 估值标准（40分）

#### 1.1.1 市盈率标准（10分）
- **优秀（10分）**：PE ≤ 10
- **良好（8分）**：10 < PE ≤ 15
- **一般（5分）**：15 < PE ≤ 20
- **较差（2分）**：20 < PE ≤ 25
- **不合格（0分）**：PE > 25 或 PE < 0

#### 1.1.2 市净率标准（10分）
- **优秀（10分）**：PB ≤ 1.0
- **良好（8分）**：1.0 < PB ≤ 1.5
- **一般（5分）**：1.5 < PB ≤ 2.0
- **较差（2分）**：2.0 < PB ≤ 2.5
- **不合格（0分）**：PB > 2.5

#### 1.1.3 综合估值标准（10分）
- **优秀（10分）**：PE × PB ≤ 15
- **良好（8分）**：15 < PE × PB ≤ 22.5
- **一般（5分）**：22.5 < PE × PB ≤ 30
- **较差（2分）**：30 < PE × PB ≤ 40
- **不合格（0分）**：PE × PB > 40

#### 1.1.4 股息收益率（10分）
- **优秀（10分）**：股息收益率 ≥ 4%
- **良好（8分）**：3% ≤ 股息收益率 < 4%
- **一般（5分）**：2% ≤ 股息收益率 < 3%
- **较差（2分）**：1% ≤ 股息收益率 < 2%
- **不合格（0分）**：股息收益率 < 1%

### 1.2 财务健康度标准（30分）

#### 1.2.1 流动比率（10分）
- **优秀（10分）**：流动比率 ≥ 2.5
- **良好（8分）**：2.0 ≤ 流动比率 < 2.5
- **一般（5分）**：1.5 ≤ 流动比率 < 2.0
- **较差（2分）**：1.2 ≤ 流动比率 < 1.5
- **不合格（0分）**：流动比率 < 1.2

#### 1.2.2 资产负债率（10分）
- **优秀（10分）**：资产负债率 ≤ 30%
- **良好（8分）**：30% < 资产负债率 ≤ 50%
- **一般（5分）**：50% < 资产负债率 ≤ 60%
- **较差（2分）**：60% < 资产负债率 ≤ 70%
- **不合格（0分）**：资产负债率 > 70%

#### 1.2.3 净资产收益率稳定性（10分）
- **优秀（10分）**：近5年ROE均 > 10%，波动 < 3%
- **良好（8分）**：近5年ROE均 > 8%，波动 < 5%
- **一般（5分）**：近5年ROE均 > 5%，波动 < 8%
- **较差（2分）**：近3年ROE均 > 3%
- **不合格（0分）**：ROE不稳定或为负

### 1.3 盈利稳定性标准（20分）

#### 1.3.1 盈利连续性（10分）
- **优秀（10分）**：近10年连续盈利
- **良好（8分）**：近8年连续盈利
- **一般（5分）**：近5年连续盈利
- **较差（2分）**：近3年连续盈利
- **不合格（0分）**：盈利不连续

#### 1.3.2 营收增长稳定性（10分）
- **优秀（10分）**：近5年营收年均增长 > 5%，无负增长年份
- **良好（8分）**：近5年营收年均增长 > 3%，负增长年份 ≤ 1年
- **一般（5分）**：近5年营收年均增长 > 0%，负增长年份 ≤ 2年
- **较差（2分）**：近3年营收年均增长 > 0%
- **不合格（0分）**：营收增长不稳定

### 1.4 成长性标准（10分）

#### 1.4.1 每股收益增长（10分）
- **优秀（10分）**：近5年EPS年均增长 > 10%
- **良好（8分）**：近5年EPS年均增长 > 5%
- **一般（5分）**：近5年EPS年均增长 > 3%
- **较差（2分）**：近3年EPS年均增长 > 0%
- **不合格（0分）**：EPS增长为负或不稳定

**格雷厄姆策略总分：100分，及格线：60分**

## 二、巴菲特质量成长策略标准

### 2.1 企业质量标准（40分）

#### 2.1.1 竞争优势（护城河）（15分）
- **优秀（15分）**：具有明显的品牌优势、专利保护或规模效应
- **良好（12分）**：在细分领域有一定竞争优势
- **一般（8分）**：竞争优势不明显但有一定壁垒
- **较差（4分）**：竞争激烈，优势微弱
- **不合格（0分）**：无明显竞争优势

#### 2.1.2 行业地位（10分）
- **优秀（10分）**：行业龙头，市场份额 > 20%
- **良好（8分）**：行业前三，市场份额 > 10%
- **一般（5分）**：行业前五，市场份额 > 5%
- **较差（2分）**：行业前十
- **不合格（0分）**：行业地位不明显

#### 2.1.3 管理层质量（15分）
- **优秀（15分）**：管理层持股 > 10%，经营稳定，无重大违规
- **良好（12分）**：管理层持股 > 5%，经营较稳定
- **一般（8分）**：管理层持股 > 1%，无重大问题
- **较差（4分）**：管理层持股较少，但无重大违规
- **不合格（0分）**：管理层存在重大问题或违规

### 2.2 财务质量标准（35分）

#### 2.2.1 净资产收益率（15分）
- **优秀（15分）**：近5年平均ROE > 20%，最低年份 > 15%
- **良好（12分）**：近5年平均ROE > 15%，最低年份 > 10%
- **一般（8分）**：近5年平均ROE > 12%，最低年份 > 8%
- **较差（4分）**：近5年平均ROE > 10%，最低年份 > 5%
- **不合格（0分）**：ROE不达标

#### 2.2.2 毛利率水平（10分）
- **优秀（10分）**：毛利率 > 50%
- **良好（8分）**：毛利率 > 40%
- **一般（5分）**：毛利率 > 30%
- **较差（2分）**：毛利率 > 20%
- **不合格（0分）**：毛利率 ≤ 20%

#### 2.2.3 现金流质量（10分）
- **优秀（10分）**：经营现金流/净利润 > 1.2，近5年均为正
- **良好（8分）**：经营现金流/净利润 > 1.0，近5年均为正
- **一般（5分）**：经营现金流/净利润 > 0.8，近3年均为正
- **较差（2分）**：经营现金流为正，但质量一般
- **不合格（0分）**：经营现金流质量差

### 2.3 成长性标准（15分）

#### 2.3.1 营收增长（8分）
- **优秀（8分）**：近5年营收年均增长 > 15%
- **良好（6分）**：近5年营收年均增长 > 10%
- **一般（4分）**：近5年营收年均增长 > 7%
- **较差（2分）**：近5年营收年均增长 > 5%
- **不合格（0分）**：营收增长 ≤ 5%

#### 2.3.2 利润增长（7分）
- **优秀（7分）**：近5年净利润年均增长 > 20%
- **良好（5分）**：近5年净利润年均增长 > 15%
- **一般（3分）**：近5年净利润年均增长 > 10%
- **较差（1分）**：近5年净利润年均增长 > 5%
- **不合格（0分）**：利润增长 ≤ 5%

### 2.4 估值合理性（10分）

#### 2.4.1 PEG指标（10分）
- **优秀（10分）**：PEG < 0.8
- **良好（8分）**：0.8 ≤ PEG < 1.0
- **一般（5分）**：1.0 ≤ PEG < 1.3
- **较差（2分）**：1.3 ≤ PEG < 1.5
- **不合格（0分）**：PEG ≥ 1.5

**巴菲特策略总分：100分，及格线：70分**

## 三、彼得·林奇成长股策略标准

### 3.1 成长性标准（40分）

#### 3.1.1 营收增长率（15分）
- **优秀（15分）**：近3年营收年均增长 > 25%
- **良好（12分）**：近3年营收年均增长 > 20%
- **一般（8分）**：近3年营收年均增长 > 15%
- **较差（4分）**：近3年营收年均增长 > 10%
- **不合格（0分）**：营收增长 ≤ 10%

#### 3.1.2 利润增长率（15分）
- **优秀（15分）**：近3年净利润年均增长 > 30%
- **良好（12分）**：近3年净利润年均增长 > 25%
- **一般（8分）**：近3年净利润年均增长 > 20%
- **较差（4分）**：近3年净利润年均增长 > 15%
- **不合格（0分）**：利润增长 ≤ 15%

#### 3.1.3 增长可持续性（10分）
- **优秀（10分）**：增长率稳定，波动 < 5%
- **良好（8分）**：增长率较稳定，波动 < 10%
- **一般（5分）**：增长率有波动，但趋势向上
- **较差（2分）**：增长率波动较大
- **不合格（0分）**：增长不可持续

### 3.2 PEG估值标准（25分）

#### 3.2.1 PEG指标（25分）
- **优秀（25分）**：PEG < 0.5
- **良好（20分）**：0.5 ≤ PEG < 0.8
- **一般（15分）**：0.8 ≤ PEG < 1.0
- **较差（8分）**：1.0 ≤ PEG < 1.3
- **不合格（0分）**：PEG ≥ 1.3

### 3.3 财务健康度（20分）

#### 3.3.1 资产负债率（10分）
- **优秀（10分）**：资产负债率 < 40%
- **良好（8分）**：40% ≤ 资产负债率 < 50%
- **一般（5分）**：50% ≤ 资产负债率 < 60%
- **较差（2分）**：60% ≤ 资产负债率 < 70%
- **不合格（0分）**：资产负债率 ≥ 70%

#### 3.3.2 净资产收益率（10分）
- **优秀（10分）**：ROE > 20%
- **良好（8分）**：15% < ROE ≤ 20%
- **一般（5分）**：12% < ROE ≤ 15%
- **较差（2分）**：8% < ROE ≤ 12%
- **不合格（0分）**：ROE ≤ 8%

### 3.4 市场地位与前景（15分）

#### 3.4.1 行业前景（8分）
- **优秀（8分）**：新兴行业，市场空间巨大
- **良好（6分）**：成长行业，市场空间较大
- **一般（4分）**：传统行业，但有创新
- **较差（2分）**：传统行业，增长有限
- **不合格（0分）**：夕阳行业

#### 3.4.2 公司规模（7分）
- **优秀（7分）**：中小型公司，市值50-200亿
- **良好（5分）**：中型公司，市值200-500亿
- **一般（3分）**：大型公司，市值500-1000亿
- **较差（1分）**：超大型公司，市值 > 1000亿
- **不合格（0分）**：微型公司，市值 < 50亿

**林奇策略总分：100分，及格线：65分**

## 四、综合评分体系

### 4.1 策略权重分配

根据当前A股市场特点，建议权重分配：
- **格雷厄姆价值投资策略**：30%
- **巴菲特质量成长策略**：40%
- **彼得·林奇成长股策略**：30%

### 4.2 综合评分计算

**综合得分 = 格雷厄姆得分 × 0.3 + 巴菲特得分 × 0.4 + 林奇得分 × 0.3**

### 4.3 投资建议分级

- **强烈推荐（85-100分）**：符合多个大师标准的优质股票
- **推荐（70-84分）**：符合部分大师标准的良好股票
- **谨慎关注（60-69分）**：有一定投资价值但需谨慎
- **不推荐（60分以下）**：不符合投资大师标准

### 4.4 特殊考虑因素

#### 4.4.1 行业加分项
- 新能源、医药生物、科技创新等国家政策支持行业：+5分
- 消费升级、服务业等内需驱动行业：+3分

#### 4.4.2 风险扣分项
- ST、*ST等风险警示股票：-20分
- 近期有重大违规或财务造假：-30分
- 主营业务不清晰或频繁变更：-10分

## 五、实施流程

1. **数据收集**：获取A股所有股票的基本信息和财务数据
2. **指标计算**：计算各项评分指标
3. **策略评分**：按三套策略分别评分
4. **综合排序**：计算综合得分并排序
5. **人工筛选**：对高分股票进行定性分析
6. **投资建议**：形成最终投资建议

此标准体系将作为后续选股分析的核心依据。
