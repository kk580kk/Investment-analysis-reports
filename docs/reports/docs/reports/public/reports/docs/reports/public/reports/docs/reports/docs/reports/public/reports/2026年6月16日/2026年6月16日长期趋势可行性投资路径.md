## 二、MECE框架分析

### 2.1 经济维度

- **居民存款大迁移：** "2万亿居民存款搬去了哪"登上热搜，意味着国内居民正在从银行存款向资本市场/消费/房产转移。结合6月15日A股券商股大涨，这一趋势正在加速形成。
- **消费降级与升级并存：** 婚宴上带冰碴预制菜（消费维权意识提升）、路虎极光L售价大跳水（高端燃油车贬值加速）、溜溜梅股价暴涨185%（AI概念"LLM"谐音带火食品股——非基本面驱动，投机性强）。
- **财富效应：** 股市上涨尤其是半导体/算力方向大面积涨停，产生局部财富效应，可能进一步引导资金入场。

**长期判断（>10年）：** 中国经济正处于"储蓄搬家"的长期趋势拐点，居民资产配置将从房产+存款，转向权益+多元化配置。这个过程至少持续10-15年。

### 2.2 产业维度

#### A. 半导体产业链——历史级投资窗口

| 子方向 | 关键事件 | 长期影响 |
|--------|---------|---------|
| 材料端 | 中国纯硅技术突破、硅基量子芯片材料攻克 | 摆脱进口依赖，实现国产替代 |
| 设计端 | 国产GPU四小龙将上市 | GPU国产化生态形成 |
| 制造端 | 粤芯半导体IPO过会 | 12英寸晶圆产能扩张 |
| AI芯片 | 燧原科技科创板IPO过会 | AI训练/推理芯片国产化 |
| 算力基建 | A股算力方向大面积涨停 | 算力即国力的共识形成 |
| PCB | 4000亿PCB龙头涨停 | 电子工业基础元器件扩产 |

**长期判断（>10年）：** 中国半导体正处于"自主可控"的10年级别上行周期。从材料到设计到制造到封测，全链条国产替代不可逆。

#### B. 人形机器人产业链

- 全球人形机器人九成中国造 → 中国掌握供应链话语权
- 人形机器人迎来新一轮大降价 → 从B端走向C端的临界点
- 京东率先采用人形机器人直播比价 → 商业化落地加速
- 理想汽车定义"具身智能汽车" → 具身智能+汽车交叉融合

**长期判断（>10年）：** 人形机器人是AI的下一个万亿级硬件载体。中国在供应链端占据绝对优势，但核心零部件（伺服电机、减速器、传感器）仍需关注国产替代进程。

#### C. 新能源汽车与智能驾驶

- 日系车巨头（推测丰田/本田）将为中国品牌代工 → 全球汽车产业权力转移标志性事件
- 理想智驾目标四季度对齐特斯拉 → 国内智驾进入L3/L4爆发前夜
- 蔚来李斌表示有望增长40-50% → 新势力进入盈利拐点
- 路虎极光L售价大跳水 → 传统豪华品牌溢价崩塌

**长期判断（>10年）：** 中国汽车产业已完成从"市场换技术"到"技术输出"的根本反转。日系车为中国品牌代工是这一历史性变化的标志性事件。

#### D. 太空经济

- SpaceX IPO募集857亿美元，市值2.1万亿美元 → 全球最大IPO，太空经济金融化
- SpaceX联手英伟达打造太空算力 → 边缘计算+卫星互联网融合
- 马斯克称SpaceX营收或近1万亿美元 → 商业航天的想象空间被打开

**长期判断（>10年）：** 太空经济正在从政府主导转向商业驱动。SpaceX IPO是太空经济的"iPhone时刻"。

### 2.3 社会维度

- **英国16岁以下全面禁用社交媒体**：可能引发全球效仿（澳大利亚已先行），对社交媒体平台、青少年消费、在线教育产生深远影响。
- **肥胖最新标准发布**：大健康/减重/医药赛道长期受益。
- **多所双一流高校宣布扩招**：高等教育普及化趋势延续，教育相关产业（住宿、培训、出版）长期受影响。

**长期判断（>10年）：** 全球社交媒体的监管收紧是大趋势，中国已先行一步。这一趋势会改变互联网流量格局和广告模式。同时，大健康产业（肥胖/代谢/老龄化）是确定的10年+赛道。

### 2.4 国际维度

- **美国伊朗达成和平协议——历史性转折**
  - 伊朗将对霍尔木兹海峡航运服务收费 → 全球能源贸易成本上升
  - 中东地缘风险溢价下降 → 原油长期价格中枢可能下移
  - 中美在中东博弈格局改变 → 中国"一带一路"中东段战略空间扩大
  - 14条款谅解备忘录细节：预计包括核问题、制裁解除、航运安全等

- **特朗普警告马克龙**：美欧贸易摩擦风险上升，影响出口导向型企业。
- **波兰地下挖出32具胎儿遗骸**：欧洲社会议题激化。
- **俄罗斯战略轰炸机坠毁**：俄军装备老化问题凸显，俄乌冲突消耗持续。

**长期判断（>10年）：** 美伊和解是中东地缘格局20年来最大变化。可能路线：美伊和解 → 以色列-阿拉伯关系进一步正常化 → 中东从"火药桶"转向"经济走廊"。霍尔木兹海峡收费将增加全球能源成本，长期利好新能源替代。

|---------|---------|---------|
| 基建"六张网" | 全国统一基础设施网络（交通/能源/水利/信息/物流/生态） | 10年基建投资纲领，类似"新基建2.0" |

| 中方制裁菲律宾防长 | 南海主权博弈 | 地缘政治持续紧张 |

**长期判断（>10年）：** "六张网"是中国基础设施的顶层设计重构，将引导未来10年财政投资方向。半导体国产化政策从资金支持转向资本市场融资支持。

**逻辑：** 2万亿存款搬家，资金从存款流向股市和优质房产。

**操作建议：**
