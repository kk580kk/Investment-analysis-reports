# 2026年5月21日中油工程投资机会 600339

## 一、公司概况

中油工程（600339.SH）是中国石油集团旗下唯一工程建设业务上市公司，主要提供油气田地面工程、管道工程、炼油化工工程、LNG接收站等领域的总承包（EPC）服务。2026年一季度主营收入31.99亿元，同比增长23.71%；归母净利润2.28亿元，同比增长22.19%。

## 二、投资逻辑

### 核心催化剂：《中俄睦邻友好合作条约》延期 + "西伯利亚力量2号"潜在推进

普京2026年5月访华期间，《中俄睦邻友好合作条约》确认延期。能源合作为中俄合作核心领域，"西伯利亚力量2号"天然气管道（年输气量500亿立方米，总长度超6000公里，估算总投资超1000亿美元）作为标志性项目被列为重点议题。

### 基本面支撑

1. **订单能见度长**：中石油集团"十五五"期间国内管道新建+更新改造投资超3000亿元，叠加海外项目（中东、中亚、俄罗斯），在收入端提供3-5年确定性
2. **一季报验证**：营收同比增23.71%，扣非净利润同比增53.3%，增长势头良好
3. **估值安全垫**：当前PE约15倍，低于国际油服工程同行（斯伦贝谢PE约25倍），具备估值修复空间

## 三、交易决策结论

**买入：中油工程（600339），买入价格4.50-5.00元/股**

- **买入依据**：当前股价对应2025年PE约15倍，处于近三年估值低位。中俄条约延期落地后，"西伯利亚力量2号"项目预期升温将推动估值修复至18-20倍。技术面看，股价在4.50元附近形成强支撑平台，成交量温和放大，主力资金近期有回补迹象。

- **持有周期**：中长期（12-18个月）
