# 2026年5月21日辽港股份中俄贸易投资机会 601880

## 一、公司概况

辽港股份（601880.SH）是大连港和营口港整合后的东北最大港口集团，主要提供油品/液体化工品码头、集装箱码头、散杂货码头、散粮码头、汽车码头、客运滚装、港口增值服务等业务。2026年一季度主营收入26.06亿元，同比上升3.1%；归母净利润3.59亿元，同比上升75.87%。

## 二、投资逻辑

### 核心催化剂：中俄贸易额持续高增长

2025年中俄贸易额达2279亿美元，2026年前4月双边贸易额达852.41亿美元，同比增19.7%。辽港股份直接受益逻辑：

1. **东北出海口地位**：大连港是东北地区最大的外贸口岸，中俄贸易增量（尤其是俄远东能源出口）的必经门户
2. **北极航道节点**：随着全球变暖，北极航道通航期逐年延长，大连港有望成为亚欧北极航线的南端核心枢纽港
3. **俄远东进口增量**：俄罗斯对华原油、液化天然气、粮食、木材等大宗商品出口增长，直接提升辽港股份散杂货及油品码头吞吐量

### 基本面支撑

1. **一季报利润大幅改善**：净利润同比增75.87%，扣非净利润同比增65.86%
2. **负债率极低**：资产负债率仅22.58%，财务健康度高
3. **估值极其安全**：当前股价约1.61元，PB仅约0.5倍，处于历史估值底部区域
4. **分红稳定**：历史分红率约40%，股息率约3%

## 三、交易决策结论

**买入：辽港股份（601880），买入价格1.50-1.65元/股**

- **买入依据**：PB仅0.5倍，为上市以来最低区域之一，安全边际充足。中俄贸易量持续增长（年增近20%）将逐步反映在港口吞吐量和盈利能力上。Q1净利润增75.87%已开始体现贸易增量效益。股价1.60元左右横盘整理近3个月，筹码集中度提升。
- **目标价格**：2.20-2.50元/股（对应PB修复至0.7-0.8倍，参考行业港口股平均PB）
- **持有周期**：中长期（12-24个月）
- **止损价格**：1.35元/股
