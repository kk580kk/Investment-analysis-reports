### 🏠 个人房产策略（25万首付方案）

**标的：** 广州/深圳核心区二手住宅（60-80㎡小户型）
- 单价区间：5-8万元/㎡，总价约300-480万
- 首付25万（约5%-8%首付，利用低首付政策）
- 贷款利率：3.06%（历史低位），月供约1.2-1.5万
- 租金覆盖比：约60%-80%（月租金6000-8000元）
- **关键判断：** 政治局定调"稳楼市"+广深限购松绑+利率历史低位+油价回落降低通胀=一线城市核心资产进入中长期配置窗口
