# 2026年06月11日雄安绿色能源与算力投资机会 300137.SZ

## 核心事件
2026年6月10日，2026雄安未来之城场景汇“雄安国际绿色能源技术应用大赛”决赛启动，聚焦绿色能源先进制造及未来场景应用。

## 未来变化预测
未来5-10年，雄安新区将成为全球领先的绿色低碳城市典范。绿色能源将与算力中心深度融合，实现“绿电供绿算”。雄安及周边区域将形成数万P规模的智能算力集群，支撑京津冀智能经济发展。

## 投资机会判断
**存在可落地的投资机会。** 先河环保近期拟向控股股东定增募资，且通过收购太行智算中心积极转型算力领域，计划2026年底规划达5万P算力，是雄安绿色能源与算力融合的典型标的。

## 投资对象描述
*   **投资对象：** 先河环保（300137.SZ）
*   **主要产品/业务：** 环境监测系统、智能算力租赁服务（H100/H200/B300系列集群）、绿色能源管理。

## 交易决策结论
*   **买入：** 先河环保（300137.SZ）
*   **买入价格：** 5.0 - 5.3 元（当前价5.20元，定增预案落地后股价具备较强支撑）。
*   **卖出：** 先河环保（300137.SZ）
*   **卖出价格：** 8.5 元（预期算力集群投产后的业绩爆发带来的估值弹性）。

## 风险点
1.  定增进度不及预期：若定增受阻，资金链及转型进度将受影响。
2.  算力租赁价格波动：全球算力供给增加可能导致租赁单价下滑。
3.  管理层变动：近期监管措施及治理结构变化可能带来短期扰动。
