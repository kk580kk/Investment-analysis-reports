| 维度 | 核心影响 |
|------|---------|
| 经济 | 半导体产业成为拉动经济增长的核心引擎，带动上下游产业链投资 |
| 产业 | AI芯片、先进封装、半导体设备等细分领域将迎来爆发式增长 |
| 社会 | AI技术加速落地，对算力基础设施提出更高要求 |
| 国际 | 中美半导体科技博弈推动中国加快构建自主可控产业链 |
| 政策 | 国家持续出台政策支持半导体产业发展 |
| 资产价格 | 半导体板块优质资产估值将持续提升 |
