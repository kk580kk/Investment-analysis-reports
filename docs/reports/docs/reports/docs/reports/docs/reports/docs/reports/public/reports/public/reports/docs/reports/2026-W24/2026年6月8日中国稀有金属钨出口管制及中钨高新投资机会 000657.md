# 2026年6月8日中国稀有金属钨出口管制及中钨高新投资机会 000657

## 1. 核心事件
2026年6月7日消息，全球最大钨生产国中国对日本的碳化钨、钨粉等关键加工品出口量已降至零，日本汽车及航空制造业深陷原材料断供困境，核心供应商产品价格最高上调六成。

## 2. 未来变化预测
未来5-10年，中国将进一步强化稀有金属的全球定价权与出口配额管理。钨作为“工业牙齿”及战略性军工材料，其全球供应链将从“全球分工”转向“资源主权化”，日本及欧美制造业将面临长期、结构性的原材料成本中枢上移，推动全球制造业向具备资源保障能力的地区加速转移。

## 3. 影响分析 (MECE规则)

| 维度 | 影响分析 |
| :--- | :--- |
| **经济** | 推动稀有金属价格进入长期上行周期，提升中国资源型出口的附加值。 |
| **产业** | 日本汽车与航天产业链因原材料成本暴涨（最高涨至3倍）面临利润挤压；国内钨加工龙头全球竞争力被动增强。 |
| **社会** | 强化资源安全意识，促进国内循环经济与二次钨回收产业的技术突破。 |
| **国际** | 中日贸易摩擦延伸至战略资源领域，全球产业链加速“去风险化”与重构。 |
| **政策** | 中国《出口管制法》及两用物项清单执行力度达到历史高点，战略储备制度进一步完善。 |
| **资产价格** | 钨矿及碳化钨价格预计在未来三年保持每年15%-20%的复利增长，资源型企业估值重塑。 |

## 4. 投资机会判断
**存在明确可执行投资机会。** 随着中国限制对日出口，全球钨资源供给缺口将长期化，国内拥有全产业链整合能力的龙头企业将受益于价格上涨与国产替代。

## 5. 投资对象描述
**中钨高新 (000657.SZ)**：中国五矿集团旗下，全球最大的硬质合金生产企业，管理着中国最大的钨矿资源储备，主要产品包括切削刀具、钻头、硬质合金等。
**厦门钨业 (600549.SH)**：全球领先的钨钼产品生产商，拥有完整的钨产业链，同时在PCB铜箔和新能源材料领域具备极高市场份额。

## 6. 投资决策结论
*   **买入：中钨高新 (000657)**
    *   **买入价格**：22.50元人民币以内（当前估值受资源重估驱动）。
    *   **入场逻辑**：资源主权化背景下，掌握上游矿山与中游精深加工的企业具备绝对定价权。
    *   **风险点**：中日关系意外缓和导致出口限制松动；全球制造业衰退压制下游刀具需求。

*   **买入：厦门钨业 (600549)**
    *   **买入价格**：65.00元人民币以内。
    *   **入场逻辑**：PCB需求爆发与钨价上行双轮驱动。
    *   **风险点**：光伏与新能源电池材料业务竞争加剧。
