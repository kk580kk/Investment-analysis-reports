# 2026年5月12日美伊战争能源投资机会

## 核心事件
美伊战争持续，霍尔木兹海峡封锁导致全球石油储备严重消耗，IEA预测2026年全球石油供应将比总需求短少每日178万桶。

## 未来变化预测
未来5-10年，地缘政治冲突频发将重塑全球能源供应链，化石能源的战略安全地位凸显，高油价可能成为常态。

## 影响分析
*   **经济**：推高全球通胀水平（美国4月CPI升至3.8%），延缓央行降息步伐。
*   **产业**：传统油气开采及油服企业盈利能力大幅提升。
*   **社会**：生活成本上升，可能引发部分地区社会动荡。
*   **国际**：中东地缘政治格局重组，大国博弈加剧。
*   **政策**：各国加速战略石油储备建设及本土能源开发。
*   **资产价格**：原油价格面临飙升风险，能源股估值修复。

## 投资机会判断
存在可落地的投资机会。尽管战事当前，能源股估值依然低廉，具备较高的安全边际和分红收益。
