# 2025年中国政府重点支持发展产业

## 来源：2025年政府工作报告

### 产业发展重点方向

1. **创新驱动发展**
   - 大力推动创新驱动发展，促进产业结构优化升级
   - 推进科技强国建设，全面启动实施国家科技重大专项
   - 加快完善重大科技基础设施体系，加强核心创新人才培养

2. **工业经济转型**
   - 稳定工业经济运行，推进制造业技术改造升级
   - 制造业投资增长9.2%，新增省一批国家级先进制造业集群
   - 高端制造、北斗应用、新型储能等新兴产业快速发展

3. **数字经济发展**
   - 制定修订环保、安全等强制性国家标准
   - 加快数字中国建设，数字经济核心产业增加值占国内生产总值比重达到10%左右

4. **区域协调发展**
   - 统筹城乡区域协调发展，优化经济布局
   - 出台实施新型城镇化战略五年行动计划
   - 扎实推进城镇更新，常住人口城镇化率提高到67%

5. **绿色低碳发展**
   - 扎实推进碳达峰碳中和，出台促进高质量分业化的政策措施
   - 强化对企业碳核算的支持，高效毕业生、脱碳人口
   - 推进教育强国建设，提高义务教育阶段家庭经济困难学生生活补助标准

### 具体产业领域

1. **新兴产业**
   - 新能源：新能源汽车年产量突破1300万辆
   - 人工智能：量子科技等领域取得新成果
   - 生物技术：生物技术、基因技术等前沿技术发展

2. **传统产业升级**
   - 钢铁、有色、石化、化工、建材等传统产业改造升级
   - 制造业重点产业链高质量发展行动

3. **服务业发展**
   - 现代服务业加快发展
   - 信息传输软件和信息技术服务业增长10.9%
   - 科技服务、商务服务等生产性服务业发展

### 政策支持措施

1. **财政政策**
   - 实施更加积极有为的宏观政策
   - 扩大国内需求，推动科技创新和产业创新融合发展

2. **投资政策**
   - 设备工器具购置投资增长15.7%
   - 家电类商品零售额增长12.3%
   - 推动房地产市场止跌回稳

3. **开放政策**
   - 扩大高水平对外开放，增强发展内生动力
   - 深化国际合作，持续优化营商环境



## 具体重点发展产业详细政策支持

### 1. 新能源产业
**政策文件**：
- 《2025年能源工作指导意见》
- 《关于大力实施可再生能源替代行动的指导意见》
- 《2024—2025年节能降碳行动方案》

**支持措施**：
- 新能源汽车年产量突破1300万辆
- 新能源汽车最高补贴1.5万元/辆
- 加快建设以沙漠、戈壁、荒漠为重点的大型风电光伏基地
- 氢能产业纳入能源管理，推进氢能领域重大技术装备发展

### 2. 半导体产业
**政策支持**：
- "十四五"规划将半导体产业列为战略性新兴产业
- 国家集成电路产业投资基金支持
- 税收优惠、研发补贴和产业基金等政策
- 2024年中国大陆半导体设备出货额将创历史新高达490亿美元

**发展重点**：
- 第三代半导体产业保持高增长态势
- 功率器件市场规模达到176亿元，同比增长14.8%
- 支持并购重组，推动产业链整合

### 3. 生物医药产业
**重要政策**：
- 《关于全面深化药品医疗器械监管改革促进医药产业高质量发展的意见》
- 各地"生物医药38条"等专项支持政策

**支持重点**：
- 创新药和医疗器械发展机制完善
- 全链条创新发展支持，从基础科学到市场应用
- 已有19款一类创新药和25款三类创新医疗器械获批上市
- 对1类创新药不同阶段给予10%-30%研发投入支持

### 4. 人工智能产业
**核心政策**：
- "人工智能+"行动持续推进
- 《北京市推动"人工智能+"行动计划（2024—2025年）》
- 各地AI产业发展专项行动方案

**发展目标**：
- 到2025年底形成3-5个先进可用、自主可控的基础大模型产品
- 100个优秀的行业大模型产品和1000个行业成功应用
- 重点推动AI+机器人、AI+汽车、AI+制造等应用落地

### 5. 高端制造业
**政策支持**：
- 制造业重点行业新购固定资产加速折旧政策
- 先进制造业企业增值税加计抵减
- 2024年支持科技创新和制造业发展的主要政策减税降费达26293亿元

**发展方向**：
- 推动制造业高端化、智能化、绿色化发展
- 建设30个国家级先进制造业集群
- 支持制造业企业向产业链上游研发设计和下游服务环节延伸

### 6. 未来产业
**重点领域**：
- 具身智能、人形机器人
- 量子信息、类脑智能
- 空天信息、低空经济
- 基因技术、未来网络
- 深海空天开发、氢能与储能

**政策特点**：
- 前瞻性布局，抢占未来产业制高点
- 各地政府工作报告普遍提及未来产业布局
- 重点支持从0到1的原始创新

## 政策支持强度评估

### 高强度支持产业（政策密集度最高）
1. 新能源（特别是新能源汽车）
2. 人工智能
3. 半导体

### 中高强度支持产业
1. 生物医药
2. 高端制造业
3. 数字经济

### 新兴重点支持产业
1. 未来产业各细分领域
2. 绿色低碳产业
3. 现代服务业与制造业融合
