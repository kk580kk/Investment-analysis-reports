# 2026年03月09日 中国"十五五"规划AI产业10万亿目标投资机会

> **报告日期**：2026年03月09日（昨日新闻）  
> **报告生成**：2026年03月10日  
> **分析师**：Manus AI  
> **影响时限评估**：超过10年（国家战略级别的产业重塑，将持续至2035年）

---

## 一、核心事件

2026年3月9日，中国"十五五"规划（2026–2030年）草案在全国两会期间发布，其中人工智能产业目标规模设定为**10万亿元**，全面实施"**人工智能+**"行动，卫星互联网首次作为独立产业单独列入规划。规划明确将AI、量子计算、类脑计算等列为"未来产业"核心方向，并提出在2030年前实现AI算力基础设施全面国产化。

与此同时，两会期间多位半导体企业家代表联合呼吁，将"建造中国的ASML"（即国产EUV光刻机）列为十五五期间的国家战略任务，寒武纪、中芯国际、北方华创等企业负责人均公开表态支持。

---

## 二、未来变化预测（5–10年结构性改变）

**AI产业规模跃升**：中国AI产业规模从2025年约3.5万亿元，在十五五期间实现约3倍增长，达到10万亿元。"人工智能+"将渗透制造、医疗、教育、金融、交通等全行业，AI成为中国经济增长的核心引擎。

**算力基础设施大规模建设**：十五五期间，中国将新建大量AI数据中心，算力规模预计翻倍，国产AI芯片（寒武纪、海光信息、摩尔线程等）将加速替代英伟达，国产服务器、存储、网络设备将迎来爆发式增长。

**半导体设备国产化加速**：EUV光刻机国产化被列为国家战略，北方华创、中微公司、拓荆科技等半导体设备企业将获得空前的政策支持和资金投入，国产替代进程将从当前的28nm节点向14nm、7nm加速推进。

**卫星互联网产业崛起**：卫星互联网首次单独列入规划，中国将加速建设自主可控的低轨卫星互联网星座（"千帆星座"等），打破SpaceX星链的垄断，形成新的太空经济产业链。

## 四、投资机会判断

**存在高确定性的长期投资机会**，国家战略级别的政策支持将持续10年以上，产业趋势清晰，受益标的明确：

1. **AI算力基础设施**：国产AI芯片（寒武纪、海光信息）、服务器（浪潮信息）、数据中心（光环新网、秦淮数据）将持续受益于算力大规模建设。

2. **半导体设备**：北方华创、中微公司、拓荆科技等国产半导体设备龙头，将在EUV光刻机国产化战略下获得空前的政策和资金支持。

3. **AI应用软件**：金山办公、科大讯飞、商汤科技等AI应用龙头，将受益于"人工智能+"在各行业的深度渗透。
