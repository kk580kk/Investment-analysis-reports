# 2026年6月8日中国央行连续19个月增持黄金及紫金矿业投资机会 601899

## 1. 核心事件
2026年6月7日国家外汇管理局数据显示，中国官方黄金储备已连续第19个月实现增持，5月末黄金储备达7496万盎司，环比增加32万盎司，增持力度呈加速趋势。

## 2. 未来变化预测
未来5-10年，全球去美元化进程将不可逆转。中国央行将持续优化储备结构，黄金在我国官方储备中的占比将从目前的不足5%逐步提升至10%以上，带动全球央行进入“黄金本位”回归期，金价长期中枢将因储备需求而系统性抬升。

## 3. 影响分析 (MECE规则)

| 维度 | 影响分析 |
| :--- | :--- |
| **经济** | 提升人民币信用背书，为人民币国际化提供坚实的实物资产支撑。 |
| **产业** | 黄金采选行业进入高毛利周期，全球优质金矿资源争夺战升级。 |
| **社会** | 民间避险情绪上升，实物黄金消费与投资需求持续旺盛。 |
| **国际** | 全球货币体系从“美元单一中心”向“多元货币+黄金”体系演变。 |
| **政策** | 国家战略储备制度从单一外汇向多元战略资源储备转型。 |
| **资产价格** | 国际金价长期看涨，金矿类上市公司进入估值与业绩双升的“戴维斯双击”。 |

## 4. 投资机会判断
**存在明确可执行投资机会。** 央行持续背书与地缘政治风险溢价将使黄金成为未来十年的核心避险资产，具备全球金矿布局的龙头企业具备极高配置价值。

## 5. 投资对象描述
**紫金矿业 (601899.SH)**：中国最大的金、铜生产商，在全球范围内拥有多座超大型金矿，具备极强的资源获取与成本控制能力。
**山东黄金 (600547.SH)**：国内领先的黄金采选企业，专注黄金主业，受益于金价上涨的弹性最大。

## 6. 投资决策结论
*   **买入：紫金矿业 (601899)**
    *   **买入价格**：29.50元人民币以内（近期受大盘调整影响出现黄金坑）。
    *   **入场逻辑**：金铜双驱动，央行增持提供长期底部支撑。
    *   **风险点**：海外矿山政治风险；美元超预期走强。

*   **买入：山东黄金 (600547)**
    *   **买入价格**：32.00元人民币以内。
    *   **入场逻辑**：黄金纯度高，对金价波动敏感度高。
    *   **风险点**：生产成本上升超预期。
