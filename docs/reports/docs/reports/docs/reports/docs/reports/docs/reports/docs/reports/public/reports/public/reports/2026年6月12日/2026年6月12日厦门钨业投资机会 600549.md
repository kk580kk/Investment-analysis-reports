# 厦门钨业（600549.SH）投资机会分析

**日期：2026年6月12日**

## 事件驱动

日本关东电化与中央硝子（占全球WF₆产能25%）7月1日起永久停产，全球半导体供应链面临WF₆硬短缺。中国掌握全球79%钨资源，厦门钨业作为钨产业链一体化龙头，直接受益于钨价上涨和定价权提升。

## 核心优势

1. 6N级高纯钨粉已通过供应链验证，WF₆核心原料供应商
2. 钨粉占六氟化钨生产成本60-70%，涨价弹性大
3. 多元化业务（能源新材料、稀土）提供安全垫
4. 估值远低于下游中船特气（PE 310倍）
