# 2026年5月24日常住地公共服务政策投资机会 600905

## 核心事件
2026年5月22-23日，国务院印发《关于推行常住地提供基本公共服务的实施意见》，逐步消除基本公共服务与户籍挂钩因素；同时，多国气象机构预测2026年将发生较强厄尔尼诺现象，今夏全国最高用电负荷将大增。

## 未来变化预测
未来5-10年，中国超3.5亿流动人口的“市民化”进程将实质性加速。基本公共服务与户籍解绑，将极大稳定流动人口的生活预期，释放庞大的内需消费潜力与住房需求。同时，随着城镇化质量的提升和极端天气频发，城市基础设施（特别是电力保障与清洁能源供应）将面临持续的扩容与升级压力。

## 影响分析
*   **经济**：流动人口预防性储蓄下降，消费意愿上升，成为拉动内需的新引擎；新型城镇化建设带动基础设施投资稳步增长。
*   **产业**：消费品、保障性住房建设、城市公共服务设施、以及绿色电力供应等产业将迎来长期稳定的需求支撑。
*   **社会**：社会公平性显著提升，阶层流动性增强，城市融入感提升，有利于社会长期和谐稳定。
*   **国际**：中国通过高质量城镇化释放内需，增强了经济内循环的韧性，降低对外部经济环境的依赖。
*   **政策**：“人地钱”挂钩政策将进一步深化，中央财政转移支付将更多向人口流入地倾斜。
*   **资产价格**：核心城市圈的优质公共事业资产（如绿电、水务）及大众消费龙头将获得稳定的业绩支撑与估值溢价。

## 投资机会判断
存在明确可落地的投资机会。新型城镇化与流动人口市民化将带来刚性的能源需求增长，叠加极端天气对电力保供的挑战，具备规模优势的清洁能源供应商将迎来长期稳健的发展机遇。

## 投资对象描述
**三峡能源（股票代码：600905）**：公司是中国长江三峡集团旗下的新能源业务战略实施主体，主营业务涵盖风电、光伏等清洁能源的开发、投资与运营。作为国内新能源发电的龙头企业，公司装机规模庞大，资产质量优良，能够有效受益于全社会用电量的长期增长及能源结构的绿色转型。

## 投资路径：上证/深证/港股股票投资
*   **方向**：做多新型城镇化与极端天气背景下的绿色电力保供核心资产
*   **标的类型**：沪市主板上市的清洁能源发电龙头企业
*   **产品/业务描述**：专注于风能、太阳能等新能源项目的开发、建设和运营，为全社会提供稳定、清洁的电力供应。
*   **入场逻辑**：常住地公共服务政策将加速3.5亿流动人口的城市化，带动城市刚性用电需求增长。同时，厄尔尼诺现象预示今夏用电负荷将创新高。三峡能源作为公用事业属性极强的绿电龙头，具备“需求刚性+高分红+稳健成长”的三重属性，是应对宏观不确定性的优质底仓资产。
*   **风险点**：极端天气导致风光资源不及预期；绿电市场化交易电价下行风险；新能源项目建设成本超预期。
