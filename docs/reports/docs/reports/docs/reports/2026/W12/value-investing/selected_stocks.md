# 各板块龙头股票筛选结果

## 筛选标准
基于投资框架的个股初选标准：
1. 市值超过100亿元
2. 行业龙头地位
3. 基本面稳健
4. 流动性良好
5. 符合政策支持方向

## 各板块龙头股票选择

### 1. 银行板块（2024年涨幅+37.08%）

**选择的龙头股票：**

1. **工商银行（601398）**
   - 市值：2.86万亿元（A股市值第一）
   - 地位：全球最大商业银行之一
   - 特点：稳健经营，高分红率
   - 2024年表现：股价创历史新高

2. **建设银行（601939）**
   - 市值：2.55万亿元
   - 地位：四大行之一
   - 特点：基建融资优势明显
   - 2024年表现：股价突破10元关口

3. **农业银行（601288）**
   - 市值：2.03万亿元
   - 地位：农村金融服务龙头
   - 特点：网点覆盖面广，三农业务优势
   - 2024年表现：持续创历史新高

### 2. 半导体板块（2024年表现强势）

**选择的龙头股票：**

   - 特点：技术最先进、规模最大
   - 业务：集成电路制造服务

2. **北方华创（002371）**

3. **韦尔股份（603501）**
