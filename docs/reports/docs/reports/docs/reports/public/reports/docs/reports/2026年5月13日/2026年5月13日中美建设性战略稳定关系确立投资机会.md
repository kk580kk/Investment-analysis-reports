# 2026年5月13日中美建设性战略稳定关系确立投资机会

## 一、 核心事件
2026年5月13日，美国总统特朗普抵达北京，开启时隔9年的美国总统访华之旅。中美两国元首确立"中美建设性战略稳定关系"新定位，中国承诺购买200架波音飞机、大豆及能源，双方就建立贸易委员会取得进展，中美关系进入新的历史阶段。

## 二、 未来变化预测
未来5-10年，中美关系将在竞争与合作并存的框架下运行，贸易战全面升级的极端风险有所降低，但科技脱钩和地缘政治博弈将持续。中国市场将进一步开放，吸引更多外资进入，同时中国企业的全球化步伐也将在可控范围内推进。

## 三、 影响分析

| 维度 | 影响分析 |
| :--- | :--- |
| **经济** | 中美贸易关系趋于稳定，全球供应链不确定性降低，有利于国际贸易和投资恢复。中国进口大豆、能源等大宗商品，有助于缓解国内通胀压力。 |
| **产业** | 中国航空运输业受益于波音飞机订单带来的机队更新；大豆进口增加利好中国食品加工业；能源进口多元化有助于保障能源安全。 |
| **社会** | 中美民间交流有望逐步恢复，留学、旅游等领域将受益；两国关系稳定有助于降低社会层面的不确定性和焦虑感。 |
| **国际** | 中美关系稳定将为全球地缘政治降温，有助于多边合作机制的恢复；但台湾问题、科技竞争等核心矛盾依然存在，国际格局仍处于深刻调整中。 |
| **政策** | 中国将进一步扩大开放，吸引外资；双方可能在非敏感领域推进关税互减，降低贸易壁垒。 |
| **资产价格** | 中美关系改善将提振A股和港股整体估值，尤其是此前受贸易战压制的出口型企业和消费类企业。 |

## 四、 投资机会判断
存在可落地的投资机会。中美关系改善是A股和港股的重要催化剂，尤其是受益于中美经贸正常化的行业和企业。

## 五、 投资对象描述
**中国国际航空（601111）**：中国最大的航空公司，主要业务包括国内及国际航空旅客运输、货物运输。中美关系改善将直接带动中美航线客流恢复，同时中国购买波音飞机将加速机队更新，降低运营成本。

## 六、 长期趋势可行性投资路径

**方向**：中美关系正常化受益/航空运输

**标的类型**：上证/深证/港股股票投资

**产品/业务描述**：国内及国际航空旅客运输、货物运输，中美航线是核心收入来源之一。

**入场逻辑**：中美峰会确立新型关系框架，中美航线有望加速恢复；高油价虽带来成本压力，但需求端的改善和机队更新将提升盈利能力；航空业处于后疫情时代复苏周期，估值处于历史低位。

**风险点**：中美关系再度恶化；高油价持续压缩航空公司利润；地缘政治事件导致航线中断。

**交易决策结论**：
买入：中国国际航空（股票代码601111），买入价格7.5元-8.0元区间（结合行业复苏逻辑和当前估值，逢低布局）；卖出：无明确卖出标的，若中美关系出现重大倒退信号，可考虑止损。
