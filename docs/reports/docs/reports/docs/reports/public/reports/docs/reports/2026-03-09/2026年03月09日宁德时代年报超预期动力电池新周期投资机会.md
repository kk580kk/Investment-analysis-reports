## 二、未来变化预测（5–10年结构性改变）

**动力电池行业进入新周期**：宁德时代内部预期，未来3–5年行业将保持同比20%–30%的复合增速。单车带电量系统性提升（增程车型电池容量中枢从40kWh上探至60–80kWh）、新能源商用车渗透率快速提升（2025年已达26.9%）将成为超越整车销量增速的核心驱动力。

**宁德时代护城河持续加深**：碳酸锂价格联动机制+上游矿产股权+邦普循环回收体系，构成穿越周期的盈利稳定性；PSL超级拉线等新一代制造工艺的大规模投产，将进一步拉大与二线企业的成本差距；2026年资本开支将进一步提升，在建产能321GWh，确保未来增长的产能保障。

**全球化布局加速**：海外市占率首次突破30%，2026年2月比亚迪海外销量占比首超50%（宁德时代是比亚迪的主要电池供应商之一），全球化将成为宁德时代未来10年增长的第二曲线。

**储能成为第二增长极**：储能电池销量120GWh，储能连续5年出货全球第一，储能毛利率26.71%高于动力电池（23.84%），随着全球储能市场的爆发式增长，储能业务将成为宁德时代的重要盈利来源。
