## 第三章 股票筛选方法与评估体系

### 3.1 筛选方法论

#### 3.1.1 初步筛选标准
1. **板块归属**：必须属于已识别的上升期板块
2. **流动性要求**：日均成交量不低于一定标准

4. **市场表现**：在所属板块内表现相对优秀

#### 3.1.2 深度评估体系

我们建立了多维度的股票评估体系：

**财务指标评估（40%权重）**：
- 估值水平：市盈率、市净率等估值指标
- 盈利能力：净资产收益率、毛利率等
- 成长性：营收增长率、净利润增长率等
- 财务健康：资产负债率、现金流状况等

**技术指标评估（30%权重）**：
- 趋势分析：移动平均线、价格相对位置
- 动量指标：相对强弱指数、成交量变化
- 波动性：历史波动率、最大回撤等

**基本面评估（30%权重）**：
- 行业地位：市场份额、竞争优势
- 技术实力：研发投入、专利数量
- 管理质量：治理结构、经营策略
- 政策受益：政策支持程度、补贴情况

### 3.2 估值方法

#### 3.2.1 相对估值法
基于行业平均估值水平，结合公司的成长性和质量因子，确定合理的估值区间。

**PE估值调整**：
- 低估值（PE<15）：给予30%估值溢价
- 合理估值（15≤PE<25）：给予20%估值溢价  
- 高估值（PE≥25）：给予10%估值溢价

**行业溢价调整**：
- 医药板块：25%政策溢价
- 新能源板块：30%政策溢价
- 机械板块：20%政策溢价
- 科技板块：35%政策溢价

#### 3.2.2 绝对估值法
采用现金流折现模型（DCF），基于公司未来3-5年的现金流预测，确定内在价值。

**非系统性风险**：
1. **行业风险**：特定行业的周期性波动或竞争加剧
2. **公司风险**：个别公司的经营风险或财务风险
3. **技术风险**：技术路线变化或产品迭代风险

- **时间止损**：持有期超过预期时间且未达到目标时考虑调整

**分散化策略**：
- **行业分散**：避免过度集中在单一行业
- **概念分散**：平衡不同概念板块的配置比例
- **市值分散**：兼顾大中小盘股票的配置

**动态监控**：
- **基本面监控**：定期跟踪公司财务状况和经营变化
- **技术面监控**：关注股价技术形态和成交量变化
- **政策面监控**：密切关注相关政策变化和市场预期
