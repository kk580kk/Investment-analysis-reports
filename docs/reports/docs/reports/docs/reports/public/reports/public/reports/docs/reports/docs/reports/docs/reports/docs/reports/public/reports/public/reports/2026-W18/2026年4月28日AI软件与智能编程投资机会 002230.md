# 2026年4月28日AI软件与智能编程投资机会

## 核心事件
2026年4月28日，工信部正式官宣“人工智能+软件”专项行动，并同步启动“模数共振”行动，明确以智能编程为突破口，推动软件开发向“智能协同”转型，全面赋能制造业等20个重点行业。

## 未来变化预测
未来5–10年，软件产业将发生根本性变革。传统的“人工主导”开发模式将被“智能协同”取代，智能编程工具、行业大模型及智能体（AaaS）将成为软件生产的新基建。同时，工业软件将深度融合AI技术，实现从辅助设计到自主优化的跨越，大幅提升制造业的生产效率与创新能力。

## 影响分析
*   **经济**：AI赋能软件将显著降低企业IT开发与运维成本，提升全要素生产率，为数字经济注入新动能。
*   **产业**：软件行业将迎来新一轮洗牌，掌握核心大模型及智能编程技术的企业将占据主导地位；传统工业软件企业若不能及时转型，将面临被淘汰的风险。
*   **社会**：程序员等IT从业者的工作模式将发生巨变，低代码/无代码开发普及，降低了软件开发的门槛，促进了全民创新。
*   **国际**：中国在“人工智能+软件”领域的发力，将加速基础软件及工业软件的国产替代进程，打破海外巨头的长期垄断，提升国家软件产业的全球竞争力。
*   **政策**：国家层面将持续出台配套政策，从资金、人才、标准等多维度支持AI软件生态建设，推动“模数共振”在各行业的落地。
*   **资产价格**：具备AI大模型研发能力及丰富行业落地场景的软件企业估值将迎来重估，成为资本市场长期追捧的核心资产。

## 投资机会判断
存在明确且可落地的投资机会。工信部专项行动为AI软件企业指明了发展方向并提供了政策背书。具备通用大模型底座、并在智能编程、工业软件等领域有深厚积累的企业，将率先受益于政策红利与产业升级带来的广阔市场空间。

## 投资对象描述
**科大讯飞（002230）**：公司是国内领先的人工智能企业，其“讯飞星火”大模型在代码生成、智能编程等领域具备较强竞争力。公司不仅在教育、医疗等垂直行业拥有丰富的落地场景，且持续加大研发投入，推动大模型在C端及B端的加速应用。2025年公司营收超270亿元，归母净利润大幅增长49.85%，展现出强劲的盈利能力与成长性。

## 投资路径与交易决策
*   **方向**：上证/深证/港股股票投资
*   **标的类型**：AI软件与大模型
*   **入场逻辑**：工信部“人工智能+软件”专项行动明确以智能编程为突破口，科大讯飞作为国内AI大模型龙头，其星火大模型在代码能力上具有显著优势，高度契合政策导向。公司2025年业绩亮眼，净利润大幅增长，且近期股价经历调整后（4月28日收盘价48.06元），估值具备吸引力，长期成长逻辑清晰。
*   **风险点**：AI大模型技术迭代不及预期；行业竞争加剧导致利润率下滑；宏观经济波动影响企业IT支出。
*   **交易决策结论**：
    买入：科大讯飞（股票代码002230），买入价格46.50元；
    卖出：科大讯飞（股票代码002230），卖出价格65.00元。
