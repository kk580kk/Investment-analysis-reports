# 2026年4月19日AI算力驱动光纤行业量价齐升投资机会

## 一、核心事件
2026年4月19日，国内光纤行业呈现出产品“量价齐升”的罕见景气态势，部分光纤品种（如G.657.A2）价格较去年同期暴涨650%，头部企业产销量同比增长数倍，在手订单已排至明年一季度，行业供需格局发生根本性反转。

## 二、未来变化预测
未来5–10年，全球AI算力基础设施建设将持续拉动高端光纤（如空芯光纤、超低损耗光纤）需求。由于光纤预制棒扩产周期长达1.5-2年且技术壁垒极高，行业将从过去的产能过剩转向长期的结构性紧缺。光纤网络将成为与电力网络同等重要的国家核心基础设施，具备核心技术和产能优势的头部企业将获得持续的超额利润，行业集中度将进一步提升。

## 三、影响分析

| 维度 | 影响分析 |
| :--- | :--- |
| **经济** | 光纤价格暴涨将显著提升相关制造企业的盈利能力，带动通信设备制造业的整体复苏，同时增加下游电信运营商和云服务商的资本开支成本。 |
| **产业** | 行业壁垒大幅提高，中小产能加速出清。具备“棒-纤-缆”全产业链一体化能力的龙头企业将主导市场定价权，推动光通信产业链向高端化、特种化演进。 |
| **社会** | 高速光纤网络的普及将加速全社会数字化转型，支撑自动驾驶、远程医疗、元宇宙等高带宽、低时延应用的全面落地，改变人类生活和工作方式。 |
| **国际** | 中国光纤企业在满足国内需求的同时，大量出口北美和东南亚，提升了中国在全球通信基础设施建设中的话语权和市场份额，但也可能面临更严格的贸易壁垒。 |
| **政策** | 符合“十五五”规划中关于加快数字中国建设和6G前瞻研究的战略导向，预计国家将出台更多政策支持光通信核心技术的自主研发和产业化。 |
| **资产价格** | 光纤光缆板块将迎来业绩与估值的双击，从传统的周期股向“周期+成长”股切换，相关上市公司股价具备长期上涨动力。 |

## 四、投资机会判断
存在明确、可落地、可执行的投资机会。光纤行业正处于新一轮强周期的起点，供需缺口短期内难以弥合，头部企业业绩确定性极高。当前市场已开始对该逻辑进行定价，但长期空间依然广阔。

## 五、投资对象描述
**长飞光纤（601869）**：全球光纤光缆行业的绝对龙头，连续九年全球销量第一。公司是国内少数能够大规模一体化开发与生产光纤预制棒、光纤和光缆的企业，掌握PCVD/OVD/VAD三大主流光棒制备工艺，光棒自给率100%。主要产品包括各种规格的通信光纤、光缆以及特种光纤，广泛应用于电信网络、数据中心和工业互联网等领域。

## 六、投资决策结论
**买入长飞光纤（601869）**
**买入价格**：30.00元 - 32.00元区间（结合近期股价走势及市场波动逢低建仓）
**入场逻辑**：AI算力爆发导致光纤需求激增，而上游光棒扩产周期长导致供给刚性，产品价格暴涨650%。长飞光纤作为全球龙头，产能最大且技术最强，将最大程度受益于本轮量价齐升的行业红利，业绩弹性巨大。
**风险点**：AI算力基础设施建设进度不及预期；海外市场贸易摩擦加剧导致出口受阻；行业内其他企业激进扩产导致未来产能再次过剩。
