# 重点股票深度研究报告

## 1. 工商银行（601398）- A+级别推荐

### 基本信息
- **股票代码**: 601398
- **当前价格**: 7.43元
- **总市值**: 26,481亿元
- **综合评分**: 18/20分（A+级别）

### 关键财务指标（2024年）
- **ROE（净资产收益率）**: 9.88%（2023年：10.66%）
- **ROA（总资产回报率）**: 0.78%（2023年：0.87%）
- **不良贷款率**: 1.34%（2023年：1.36%，持续改善）
- **拨备覆盖率**: 214.91%（2023年：213.97%）
- **核心一级资本充足率**: 14.10%（2023年：13.72%）
- **资本充足率**: 19.39%（2023年：19.10%）
- **成本收入比**: 29.43%（2023年：28.28%）

### 投资亮点分析
1. **估值极具吸引力**
   - PE比率仅7.87倍，远低于市场平均水平
   - PB比率0.72倍，处于破净状态
   - 符合格雷厄姆价值投资的所有标准

2. **资产质量持续改善**
   - 不良贷款率从1.36%降至1.34%，呈下降趋势
   - 拨备覆盖率超过200%，风险缓冲充足
   - 资本充足率19.39%，远超监管要求

3. **盈利能力稳健**
   - ROE接近10%，在银行业中表现优秀
   - 净利润规模庞大，盈利稳定性强
   - 成本控制能力良好

4. **系统重要性银行地位**
   - 全球最大银行之一
   - 在中国金融体系中具有核心地位
   - 享受政策支持和监管保护

### 投资风险
1. **利率环境压力**
   - 净息差面临收窄压力
   - ROE和ROA较去年有所下降

2. **经济周期影响**
   - 不良贷款可能随经济波动而变化
   - 信贷投放受宏观政策影响

### 投资评级：强烈推荐（A+）
基于格雷厄姆价值投资理论，工商银行具备极佳的投资价值：
- 估值严重低估（破净+低PE）
- 财务健康度优秀
- 分红稳定可靠
- 适合长期价值投资
