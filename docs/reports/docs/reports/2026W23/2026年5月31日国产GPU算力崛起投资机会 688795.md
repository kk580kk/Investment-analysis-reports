# 2026年5月31日国产GPU算力崛起投资机会 688795

## 一、核心事件
2026年5月31日，上海证券交易所与中证指数有限公司决定调整科创50指数样本，国产GPU领军企业**摩尔线程（688795）**及**沐曦股份（688802）**正式被调入指数，于2026年6月12日收市后生效 [1]。这一变动标志着国产高性能算力芯片已从技术攻关期进入资本市场主流赛道，获得核心权重的官方认可。

## 二、未来变化预测
在未来5-10年内，全球算力格局将发生结构性重塑。随着生成式人工智能（AGI）向物理世界渗透，算力将成为等同于电力的社会基础能源。中国将建成完全自主可控的“全栈国产算力生态”，国产GPU有望占据国内新增AI算力市场的60%以上份额，彻底摆脱对外部供应链的依赖。这一趋势将深刻影响国家战略安全与经济发展 [2]。

## 三、影响分析
| 维度 | 影响描述 |
| :--- | :--- |
| **经济** | 算力投资将取代传统基建成为拉动GDP的新引擎，高附加值的半导体产业比重显著提升，推动经济结构优化升级。 |
| **产业** | 形成“国产GPU+国产算力中心+国产大模型”的闭环生态，推动万亿级人工智能应用在各行各业落地生根，加速产业智能化转型。 |
| **社会** | 算力成本的显著下降（预计50%以上）将带动自动驾驶、智慧医疗、智能教育等普惠性服务普及，大幅提升全社会生产效率和生活品质。 |
| **国际** | 强化中国在科技博弈中的战略底气，有效打破高性能芯片领域的国际禁运与技术封锁，重塑全球科技供应链的多元化与韧性。 |
| **政策** | 国家将持续通过“未来产业”专项资金、税收优惠、政府采购等一揽子政策，向国产GPU及相关产业链倾斜，提供强有力的政策保障。 |
| **资产价格** | 硬科技资产在A股市场的权重将持续提升，具备全球竞争力的国产半导体企业，特别是GPU领域的创新型公司，将迎来估值体系的重构与价值重估。 |

## 四、投资机会判断
**存在明确可执行的投资机会。** 随着摩尔线程等企业进入核心指数，被动型基金（如科创50 ETF）将产生大规模强制性买入需求，为股价提供坚实支撑。同时，国产GPU在政务、能源、金融等关键领域的国产化替代进程将加速公司业绩兑现，形成长期增长动力。

## 五、投资对象描述
**摩尔线程（688795）**是中国领先的全功能GPU芯片设计公司，其主要产品和业务涵盖了从底层GPU芯片设计到上层软硬件生态的构建。具体而言，其产品线覆盖AI模型训练、推理、3D图形渲染以及超大规模算力集群解决方案。直观感知上，摩尔线程的GPU产品性能已能有效支撑国内主流大模型的日常迭代与优化，并广泛应用于国产新能源汽车的智能座舱与自动驾驶系统，是数字经济时代的关键基础设施提供商。

## 六、执行决策
*   **交易建议**：**买入** 摩尔线程（688795）
*   **买入价格**：**680.00元人民币**（建议在指数生效前的震荡区间分批建仓，或等待市场回调至此价格区间）
*   **入场逻辑**：指数调入带来的流动性溢价与市场关注度提升，叠加国产算力替代的确定性长期增长趋势。
*   **风险点**：先进制程代工仍面临国际地缘政治风险与技术壁垒；国际GPU巨头可能采取降价策略进行市场竞争，对公司盈利能力造成压力。
