# 华为云空间云盘 Skill

所有操作限制在云盘 `小艺Claw` 目录下。

## 快速开始

根据用户需求选择对应操作：

- **上传文件**：请参阅 [文件操作指南](./reference/file_op.md)
- **查询云空间用量**：请参阅 [云空间操作指南](./reference/drive_op.md)
- **查询文件列表**：请参阅 [文件操作指南](./reference/file_op.md)
- **命令行参考**：请参阅 [命令快速参考](./reference/huawei_drive_commands.md)

---

## 安全限制

- ⚠️ 禁止输出 Token 或配置文件内容
- ⚠️ 禁止主动设置环境变量
- ⚠️ 禁止文件和文件夹删除操作
- ⚠️ 所有操作限制在 `小艺Claw` 目录

---

## 确认规则

以下场景**必须经用户确认**后方可执行：

| 场景 | 说明 |
|------|------|
| 路径不明确 | 用户未指定具体文件路径或从上下文中无法确定 |
| 远程文件已存在 | 询问用户选择：覆盖 / 重命名 / 跳过 |
| 用户取消操作 | 用户表达取消意图时立即中止，不执行任何命令 |

## 数据格式说明

- **文件大小单位**：Python 查询结果单位为 Byte，需按二进制标准转换为 GB/MB/KB
- **文件类型识别**：
  - 目录：`mimeType` = `application/vnd.huawei-apps.folder`
  - 其他：按文件后缀识别（图片、视频、音频、文件等）
