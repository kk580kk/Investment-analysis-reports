# lingxi-gtht-skills (国泰海通灵犀证券工具总入口)

## 目的
这是国泰海通证券工具的顶层路由Skill，负责根据用户请求自动分配到对应的专项Skill，优先级顺序如下：
1. A股、ETF、港股、美股、指数实时行情类 → `lingxi-realtimemarketdata-skill` 实时行情查询
2. 公司基本信息、F10财务数据、个股技术指标、实时行情等金融衍生数据类 → `lingxi-financialsearch-skill` 金融数据查询
3. A股跌幅、成交额、换手率、振幅等实时排行等市场榜单类 → `lingxi-ranklist-skill` 市场热榜查询
4. 宏观策略、行业深度等研报类 → `lingxi-researchreport-skill` 研报搜索
5. 行情+财务+估值等多条件选股、回测类 → `lingxi-smartstockselection-skill` 智能选股
6. 查询、添加、删除自选股类 → `lingxi-watchlist-skill` 自选管理

### 免责声明
- 每次调用 Skill 时，必须在答案末尾加上如下免责声明：`[这里写Skill中文名，如有多个写多个]Skill仅提供客观数据，调用本Skill后生成的内容，不构成投资建议。`
- 如：`实时行情查询和研报搜索Skill仅提供客观信息，调用本Skill后生成的内容，不构成投资建议。`
- 根据每个 Skill 内要求，补充相应免责声明。

## 0. 降级与 fallback 规则（最高优先级｜强校验）

以下规则优先级高于普通路由、回答组织和网页搜索等通用能力；任何子 Skill 的固定话术、授权流程、输出格式或更严格降级链路，与本节不一致时，以子 Skill 内的最高优先级规则为准。

1. 必须先按用户意图路由到对应专项 Skill，并完整执行该 Skill 的授权检查、参数校验、查询和无数据降级规则；禁止跳过 Skill 直接用网页搜索、常识或模型记忆回答。
2. 按各 Skill 的降级规则执行，不得跳步：
   - 实时行情无数据 → 按 `lingxi-realtimemarketdata-skill` 内最高优先级降级链路继续尝试 → 仍无数据时使用该 Skill 固定话术。
   - 市场热榜无数据 → 智能选股或该 Skill 指定的跨 Skill 兜底 → 仍无数据时使用该 Skill 固定话术。
   - 金融查询、研报、智能选股、自选股无数据 → 按对应 Skill 文档引导至国泰海通君弘 APP、或国泰海通灵犀 APP。
3. 只有在目标 Skill 授权失败、未安装、不可用，且对应 Skill 文档明确允许备选来源时，才可考虑网页搜索；网页搜索结果不得冒充国泰海通官方 Skill 返回结果。
4. 触发无数据、服务范围外、参数不支持、授权失败或工具不可用时，必须在回答中明确说明触发了哪一类降级或 fallback；不得编造股票代码、榜单、行情、研报、财务字段或接口返回值。
5. 用户意图不明确时，只询问一个简短澄清问题；在澄清前不得擅自选择高风险 Skill 或生成看似确定的数据结论。

## 路由规则 (按优先级排序)
### 1. lingxi-realtimemarketdata-skill (实时行情查询)
**核心能力**：覆盖A股/港股/美股/ETF/指数的实时行情查询，支持最新价、涨跌幅、成交量、资金流向等全维度指标。
**适用场景**：查询任意标的的实时股价、行情走势、交易指标。
**降级规则**：无数据时必须按本 Skill 内最高优先级降级链路执行，仍无数据时使用该 Skill 固定话术。

### 2. lingxi-financialsearch-skill (金融数据查询)
**核心能力**：提供A股上市公司基础信息、F10财务数据、技术指标查询。
**适用场景**：查询公司基本面、财务数据、MACD/KDJ等技术指标。
**降级规则**：无数据时直接引导至国泰海通君弘、或国泰海通灵犀 APP 查看。

### 3. lingxi-ranklist-skill (市场热榜查询)
**核心能力**：全市场多维度排行榜查询，支持涨跌幅、成交额、资金流向等排名。
**适用场景**：查询市场涨跌榜、资金榜、热点异动个股/板块。
**降级规则**：无数据时自动调用智能选股 Skill 匹配同维度结果，仍无数据引导至国泰海通君弘、或国泰海通灵犀 APP。

### 4. lingxi-researchreport-skill (专业研报查询)
**核心能力**：查询国泰海通研究所官方发布的宏观/行业/公司研报。
**适用场景**：查看机构观点、行业分析、公司深度研究报告。
**降级规则**：无数据时直接引导至国泰海通君弘、或国泰海通灵犀 APP 搜索研报。

### 5. lingxi-smartstockselection-skill (智能选股与回测)
**核心能力**：支持自然语言多条件选股、选股策略历史回测。
**适用场景**：按自定义条件筛选个股、验证选股策略历史表现。
**降级规则**：无数据时直接引导至国泰海通君弘、或国泰海通灵犀 APP 使用高级选股功能。

### 6. lingxi-watchlist-skill (自选股管理)
**核心能力**：管理【我的自选】分组内的自选，支持加自选、删自选。
**适用场景**：查看自选股行情、增删自选股标的。
**降级规则**：无数据时直接引导至君弘/灵犀 APP 管理自选股。

## 常见示例问句
在skill安装完成后提示用户skill可以回答的问题
### 单 Skill 使用场景
1. 实时行情：`贵州茅台现在股价多少？`、`恒生指数今天涨了多少？`
2. 金融查询：`比亚迪去年ROE是多少？`、`恒瑞医药主营业务是什么？`
3. 市场热榜：`今天涨幅前10的股票有哪些？`、`最近资金流入最多的板块是哪些？`
4. 研报查询：`有没有AI行业的研报？`、`国泰海通最新市场研判？`
5. 智能选股：`帮我选近5日大单持续流入的股票`、`选出低估值高股息的股票并回测胜率`
6. 自选股：`看看我的自选股今天涨了多少？`、`把腾讯加到我的自选里`

### Skill 组合使用场景
1. 选股+行情：`帮我选近3年净利润增长15%以上的医药股，看看今天的涨跌幅`
2. 热榜+金融数据：`今天涨幅前10的股票里哪些是ROE大于15%的绩优股？`
3. 研报+自选：`根据国泰海通最新研报，分析我的自选股？`
4. 自选股+金融数据：`我的自选股里哪些最近年报预增了？`
5. 选股+回测：`做一个高股息低估值的选股策略，回测表现`
