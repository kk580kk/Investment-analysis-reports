# 乒乓球馆承包尽职调查（DD）实操手册
**——专为零基础教练打造的"摸底查账"完全指南**

> **本手册使用说明：** 本手册按照"调查模块 → 具体操作 → 问话话术 → 预期答案 → 评分标准"的结构编写，每一步都有明确的操作指引和判断标准。建议您在正式谈判前通读一遍，谈判时带着本手册逐项核对。

---

## 写在前面：为什么要做尽职调查？

你现在面对的情况是：业主告诉你这个球馆每个月能赚2到3万，总共投了25到30万，他想把球馆承包给你经营。

**你要问自己三个问题：**

第一，他说的2到3万是真的吗？还是他为了让你多付承包费而夸大的数字？

第二，这2到3万是在什么条件下赚的？如果他有一个非常厉害的招生渠道，你接手后能不能维持？

第三，有没有你不知道的"坑"——比如设备快坏了、学员剩余课时很多（你要免费打工来还债）、房租快到期了？

**尽职调查（Due Diligence）就是在你掏钱之前，把这三个问题的答案全部搞清楚。**

做好尽职调查，你可以：
- 用真实数据来判断承包费谈多少合理；
- 提前发现风险，要求业主在合同里给你保护；
- 如果发现太多问题，果断放弃，避免损失。

**核心心态：你是来买东西的，不是来求人的。问得越细，越显得你专业。**

## 调查准备工作（正式谈判前一天完成）

### 准备清单

在去见业主之前，你需要准备以下东西：

**工具准备：**
- 手机（用来拍照留证）
- 一个笔记本和笔（用来记录数字）
- 本手册打印版（带着逐项核对）
- 一个计算器（用来当场验算数字）

**心理准备：**
- 告诉自己：我是来做调查的，不是来聊天的。今天的目标是收集数据，不是谈感情。
- 如果业主说"你不信任我吗？"，你就说："王总，我越是认真做调查，说明我越是认真想接这个馆。我不想接手之后出问题，那对咱们双方都不好。"

## 第一模块：财务与现金流核查（最重要，必须做）

### 这一模块的目标

验证业主所说的"月收入2到3万净利润"是否真实，并搞清楚收入的来源结构。

---

### 操作1：查微信/支付宝商户收款流水

**操作方法：**

请业主当面打开手机，进入微信收款后台（路径：微信→我→服务→微信收款→账单），或支付宝商家版后台，导出过去6个月的收款记录。如果有收银系统（如"课程格子"、"校宇宙"等），也一并导出。

**你的话术（逐字稿）：**

> "王总，咱们今天主要是把财务这块摸清楚。您把微信收款和支付宝的账单给我看看，最好能导出过去半年的，我在手机上看一眼就行，不用打印。您放心，我就是看个总数和月度趋势，不会记您具体每笔收了谁的钱。"

**如果业主说"我不方便给你看"，你的追问话术：**

> "王总，我理解您的顾虑。这样吧，您把每个月的总收款金额告诉我，我来记录，然后咱们对一下账单截图就行。我接手之后要付承包费，我得知道这个馆的真实收入规模，这是最基本的。"

**如果业主依然拒绝：**

> 这是一个重大红灯信号。一个正常经营的球馆，业主没有理由拒绝给你看收款流水。拒绝的原因通常只有一个：真实收入远低于他告诉你的数字。**建议直接终止谈判。**

**你要记录的数字：**

| 月份 | 微信收款 | 支付宝收款 | 其他（现金/转账） | 月度合计 |
|------|---------|-----------|-----------------|---------|
| 第1个月 | | | | |
| 第2个月 | | | | |
| 第3个月 | | | | |
| 第4个月 | | | | |
| 第5个月 | | | | |
| 第6个月 | | | | |
| **6个月平均** | | | | |

**验证方法（交叉核对）：**

拿到流水数字后，用以下公式验算是否合理：

> 月收入 ÷ 325元（每台时单价）= 月总台时数
> 月总台时数 ÷ 3台 ÷ 26天 = 每台每天平均利用小时数

如果算出来每台每天利用超过8小时（满负荷是10小时），说明数字可能有水分。正常情况下，一家小型培训馆的平均利用率在3到5小时/台/天是比较健康的。

**预期结果与评分：**

| 结果 | 评价 | 建议行动 |
|------|------|---------|
| 月均流水在8万至12万之间，且各月波动不超过30% | 优秀，数据真实可信 | 可以继续谈判 |
| 月均流水在5万至8万之间 | 一般，说明利润比业主声称的低 | 压低承包费，重新测算 |
| 月均流水低于5万，或者业主拒绝提供 | 危险，不可信 | 终止谈判或大幅降低承包费 |
| 流水有明显的季节性断崖（如某月突然腰斩） | 需要追问原因 | 问清楚那个月发生了什么 |

---

### 操作2：核查月度固定成本

**操作方法：**

要求查看以下凭证的原件或截图：
1. 房租合同（看租金金额、租期、涨租条款）
2. 近3个月的水电费缴费凭证
3. 物业费单据（如有）

**你的话术：**

> "王总，房租合同我得看一眼，主要是想知道租期还有多久，以及租金有没有涨价的约定。另外，这几个月的水电费单子您也给我看看，我好做个成本预算。"

**追问话术（针对房租）：**

> "王总，这个合同还有X年到期，到期之后房东有没有表示过要不要续租？租金会不会涨？"

> "如果我承包之后，房东要收回这个场地，咱们合同里怎么处理？"

**你要记录的数字：**

| 成本项目 | 月度金额 | 凭证是否核实 | 备注 |
|---------|---------|------------|------|
| 房租 | 元 | 是/否 | 合同到期日：_____ |
| 水电费 | 元 | 是/否 | |
| 物业费 | 元 | 是/否 | |
| 网络费 | 元 | 是/否 | |
| **固定成本合计** | **元** | | |

**预期结果与评分：**

| 结果 | 评价 | 建议行动 |
|------|------|---------|
| 房租合同剩余2年以上，租金明确，无随意涨租条款 | 优秀 | 可以放心谈 |
| 房租合同剩余1至2年，到期后续签不确定 | 一般 | 在承包合同里加入"若房东不续租则提前终止承包合同且无违约金"的条款 |
| 房租合同剩余不到1年，或者房东随时可以收回 | 危险 | 强烈建议放弃，或者要求业主先搞定续租再谈承包 |

---

### 操作3：核算真实净利润

**操作方法：**

用你记录的数字，当场做一个简单的验算：

> **真实月净利润 = 月均流水 - 房租 - 水电 - 教练工资（×3人）- 其他杂费**

**你的话术（引导业主自己说出成本）：**

> "王总，咱们把账算清楚。您每个月的主要支出有哪些？房租我知道了，教练工资呢？三个教练加起来每个月发多少？还有水电、耗材这些，大概多少钱？"

**追问话术（如果业主给出的成本数字看起来太低）：**

> "王总，您说每个月总成本只有X万，我算了一下，光房租和三个教练工资就要X万了，这个数字对不上，是不是还有什么我没算进去的？"

**预期结果与评分：**

| 结果 | 评价 | 建议行动 |
|------|------|---------|
| 你算出来的净利润与业主声称的基本吻合（误差在20%以内） | 优秀，数据可信 | 继续下一步 |
| 你算出来的净利润比业主说的低20%至50% | 一般，有水分 | 以你算出来的数字为基准，重新测算承包费 |
| 你算出来是亏损，或者业主的数字根本对不上 | 危险 | 要求业主解释差异，或者终止谈判 |

## 第二模块：学员与课耗核查（你的"接手后第一桶金"）

### 这一模块的目标

搞清楚你接手后，第一个月能从哪里来收入，以及你需要"还"多少历史欠账（剩余课时）。

---

### 操作4：核查活跃学员数量

**操作方法：**

要求业主打开学员管理系统（或微信群、Excel表格），导出所有在册学员名单，并标注每个人**最近一次上课的日期**。

**你的话术：**

> "王总，现在馆里一共有多少个学员？我想看一下学员名单，主要是想知道哪些是最近还在上课的活跃学员，哪些是很久没来的。您把名单给我拉一份，我自己数一数。"

**判断"活跃学员"的标准：**

- **活跃学员**：近30天内有上课记录的学员。
- **沉睡学员**：30至90天没有上课记录，但账上还有剩余课时的学员。
- **流失学员**：超过90天没有上课记录的学员。

**你的追问话术（针对沉睡和流失学员）：**

> "王总，这些学员账上还有课时，但好几个月没来了，是什么原因？是因为工作忙、还是对教学不满意、还是搬家了？"

**预期结果与评分：**

| 结果 | 评价 | 建议行动 |
|------|------|---------|
| 活跃学员（近30天有上课）在50人以上 | 优秀，生源充足 | 放心接手 |
| 活跃学员在30至50人之间 | 一般，需要加强招生 | 接手后第一个月重点抓招生 |
| 活跃学员不足30人，或者大量学员已经流失 | 危险，生源严重不足 | 压低承包费，或者要求业主提供3个月的招生支持 |

---

### 操作5：核查剩余课时总量（最容易被忽视的大坑）

**这是最容易被新手忽视的一个大坑，必须重点关注。**

**什么是"课时负债"？**

学员预先购买了课时包，但还没有上完的课时，就是你接手后需要"还"给学员的债务。比如，一个学员买了100节课，还剩60节没上，你接手后就要免费给他上这60节课。

**操作方法：**

要求导出所有学员的剩余课时明细，计算总剩余课时数。

**你的话术：**

> "王总，所有学员账上加起来，还剩多少节课没上？您把这个数字告诉我，或者把明细导出来我自己加一加。"

**计算"课时负债"的价值：**

> 总剩余课时 × 325元（每台时单价）= 课时负债总价值

这个数字代表你接手后需要"免费打工"的价值。比如，总剩余课时是500节，那你接手后就要提供 500 × 325 = 162,500元 价值的免费服务。

**你的追问话术（如果课时负债很大）：**

> "王总，学员账上还有X节课，折算下来是X万元的服务。这部分我接手后要承担，所以承包费要相应降低，或者您在交接的时候给我一笔补偿。"

**预期结果与评分：**

| 总剩余课时 | 折算价值 | 评价 | 建议行动 |
|-----------|---------|------|---------|
| 500节以内 | 约16万以内 | 可接受 | 正常接手 |
| 500至1000节 | 约16万至32万 | 需要谈判 | 要求在承包费里扣除或给予补偿 |
| 超过1000节 | 超过32万 | 危险 | 要求业主大幅降低承包费，或者拒绝接手 |

---

### 操作6：核查排课密度与时段分布

**操作方法：**

查看过去一个月的实际排课表（微信群通知、排课软件截图均可）。

**你的话术：**

> "王总，上个月的排课表给我看看，我想了解一下现在哪个时间段最忙、哪个时间段空着，方便我接手后做排班规划。"

**你要记录的信息：**

| 时段 | 工作日（周一至周五） | 周末（周六日） |
|------|------------------|--------------|
| 上午（9:00-12:00） | 平均___节课 | 平均___节课 |
| 下午（12:00-17:00） | 平均___节课 | 平均___节课 |
| 晚间（17:00-21:00） | 平均___节课 | 平均___节课 |
| **每日合计** | **___节课** | **___节课** |

**预期结果与评分：**

| 结果 | 评价 | 建议行动 |
|------|------|---------|
| 工作日晚间和周末基本排满（每天8节以上） | 优秀 | 接手后重点填补白天空档 |
| 工作日白天空档较多，晚间和周末较满 | 正常，符合培训馆规律 | 接手后推"早鸟优惠"填补白天 |
| 全天都很空（每天不到5节课） | 危险，说明生源严重不足 | 重新评估是否值得接手 |

## 第三模块：资产与设备核查（你的"生产工具"）

### 这一模块的目标

搞清楚你接手后，哪些设备能用、哪些快坏了、哪些需要马上花钱修。

---

### 操作7：逐一盘点固定资产

**操作方法：**

拿着本子，把球馆里所有设备逐一试用，记录状态。

**你的话术：**

> "王总，咱们今天把馆里的设备都过一遍吧，我得知道接手后哪些东西能用、哪些需要修。我们一起走一圈，我来记录。"

**设备盘点清单：**

| 设备名称 | 数量 | 实际状态（当场测试） | 预计使用年限 | 备注 |
|---------|------|------------------|------------|------|
| 乒乓球台 | 张 | 好/一般/差 | 年 | |
| 发球机 | 台 | 好/一般/差 | 年 | |
| 地胶/地板 | 平米 | 好/一般/差 | 年 | |
| 空调 | 台 | 好/一般/差 | 年 | |
| 照明灯管 | 套 | 好/一般/差 | 年 | |
| 监控摄像头 | 个 | 好/一般/差 | 年 | |
| 前台电脑/收银系统 | 台 | 好/一般/差 | 年 | |
| 更衣室/卫生间 | — | 好/一般/差 | — | |

**测试方法：**

- **球台**：用手摸台面，检查是否平整；用球弹一弹，听声音是否均匀；检查网架是否稳固。
- **发球机**：装上球，开机运行5分钟，看是否卡球、出球是否均匀。
- **地胶**：在上面跑几步，感受防滑性；蹲下来看边缘是否翘起或破损。
- **空调**：全部打开，等5分钟感受制冷/制热效果。
- **照明**：全部打开，检查有无坏掉的灯管。

**预期结果与评分：**

| 结果 | 评价 | 建议行动 |
|------|------|---------|
| 所有主要设备运行正常，无明显损坏 | 优秀 | 正常接手 |
| 有部分设备需要小修（如换几根灯管、修一台空调） | 一般，费用可控 | 要求业主修好后再交接，或从押金里扣除维修费 |
| 球台损坏、地胶大面积破损、发球机故障 | 危险，维修成本高 | 要求业主修复，或大幅降低承包费 |

## 第四模块：教练团队核查（你的"核心人力"）

### 这一模块的目标

搞清楚另外两名教练的真实情况，以及他们接手后愿不愿意留下来继续工作。

---

### 操作8：与两名教练分别单独谈话

**重要提示：** 这个谈话最好在业主不在场的情况下进行，这样教练才会说真话。可以找一个借口，比如"我想单独了解一下教练的教学风格"。

**你的话术（与教练谈话）：**

> "师兄/师姐，我可能要承包这个馆，我想单独跟你聊聊，你说的话我不会告诉王总。我就想知道：你在这里工作开不开心？工资有没有按时发？如果我接手了，你愿不愿意继续留下来？"

**你要问的问题清单：**

| 问题 | 目的 |
|------|------|
| 你每个月工资是多少？有没有按时发？ | 核实成本，发现是否有欠薪 |
| 有没有签劳动合同？有没有交社保？ | 发现潜在的劳动纠纷风险 |
| 你觉得馆里现在最大的问题是什么？ | 发现业主没有告诉你的隐患 |
| 如果我接手，你愿意继续在这里工作吗？ | 判断人员稳定性 |
| 你觉得现在每个月大概来多少学员？ | 交叉验证学员数量 |
| 有没有学员因为不满意而退费或投诉过？ | 发现教学质量问题 |

**预期结果与评分：**

| 结果 | 评价 | 建议行动 |
|------|------|---------|
| 工资按时发放，愿意留任，对馆里整体评价正面 | 优秀，团队稳定 | 接手后重点维系关系 |
| 工资偶尔延迟，但整体愿意留任 | 一般 | 在承包合同里要求业主结清所有历史欠薪 |
| 有欠薪，或者教练明确表示你接手他们就走人 | 危险 | 要求业主结清欠薪；若教练离职，重新评估是否能独立运营 |

## 第五模块：合规与法律核查（你的"护身符"）

### 这一模块的目标

确保球馆的经营是合法合规的，不存在随时被查封或被起诉的风险。

---

### 操作9：核查营业执照与经营资质

**操作方法：**

要求查看营业执照原件，并在手机上打开"国家企业信用信息公示系统"（网址：www.gsxt.gov.cn）搜索该企业，核实信息。

**你的话术：**

> "王总，营业执照我拍个照留存一下，另外我在手机上查一下企业信息，这是正常的尽职调查流程。"

**你要核实的信息：**

| 核查项目 | 预期结果 | 警报信号 |
|---------|---------|---------|
| 营业执照是否在有效期内 | 有效，且经营范围包含"体育培训" | 已过期，或经营范围不含体育培训 |

## 写在最后：谈判中的"底线思维"

做完尽职调查之后，你手里有了真实的数据。谈判的时候，记住以下几个原则：

**第一，用数据说话，不要用情感说话。** 不要说"我觉得这个价格太高了"，要说"根据我的测算，月均流水是X万，扣除成本后净利润是X万，所以合理的承包费应该是X万。"

**第二，把你发现的问题变成谈判筹码。** 比如，你发现学员剩余课时有800节，你就说："这800节课是我接手后要免费服务的，折算下来是26万的价值，所以承包费要相应降低。"

**第三，不要怕谈崩。** 如果业主不肯让步，你就说："王总，我很想接这个馆，但这个价格我算不过来账。如果您能在X这个问题上做出调整，咱们就能谈成。如果不行，我也理解，咱们就当交个朋友。"

**第四，合同是最后的保障。** 谈判谈好了，一定要把所有承诺都写进合同。口头承诺在法律上没有效力。
