## 选股方法论

本报告采用三维量化评分体系：

1. **格雷厄姆价值投资评分**（满分8分）：基于PE、PB及其乘积
2. **巴菲特质量成长评分**（满分6分）：基于估值合理性与企业规模
3. **彼得·林奇成长股评分**（满分6分）：基于PEG比率与成长性

综合总分满分20分，评级标准：
- A+（强烈推荐）：15–20分
- A（推荐）：12–14分
- B+（中性偏多）：9–11分
- B（中性）：6–8分
- C（谨慎）：3–5分
- D（回避）：0–2分
