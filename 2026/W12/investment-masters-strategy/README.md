## 选股方法论

### 格雷厄姆标准
- PE ≤ 15 倍，PB ≤ 1.5 倍，PE×PB ≤ 22.5
- 流动比率 ≥ 2.0，资产负债率 ≤ 50%
- 过去十年持续盈利，EPS 增长 ≥ 1/3

### 巴菲特标准
- ROE 长期稳定 ≥ 15%
- 强大护城河（品牌、网络效应、转换成本、成本优势）
- 充裕自由现金流，低有息负债

### 彼得·林奇 GARP 标准
- 年均盈利增长率 15%–50%
- PEG ≤ 1.0（理想 ≤ 0.5）
- 业务简单易懂，财务健康
