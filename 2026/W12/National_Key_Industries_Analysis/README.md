## 报告概述

本报告基于国家发展重点支持产业政策，对上证和深圳交易所的5418只股票进行了全面筛选和深度分析。通过系统性的研究方法，识别出六大重点板块中的优质投资标的，预期3年期年化收益率可达15-35%。

分析框架以《关于推动未来产业创新发展的实施意见》（2024年1月）为政策依据，聚焦未来制造、未来信息、未来材料、未来能源四大核心板块。
