## 分析方法摘要

采用三层评分体系，从5,235只A股中系统筛选：

| 评分维度 | 满分 | 核心标准 |
|:---|:---:|:---|
| 格雷厄姆价值投资 | 3分 | PE≤15、PB≤1.5、PE×PB≤22.5 |
| 巴菲特质量成长 | 3分 | 总市值≥100亿、PE≤25、PB≤5 |
| 彼得·林奇成长股 | 3分 | PEG≤1.0得2分，PEG≤1.5得1分 |

综合评分 = 三项之和，满分9分。
